2 正交矩阵：完美的变换

2.1 定义

\(\mathbf{Q} \in \mathbb{R}^{n \times n}\) 是正交矩阵，当且仅当列向量标准正交，即 \(\mathbf{Q}^\top\mathbf{Q} = \mathbf{I}\)。

几何上它表示旋转和/或反射——形状和大小完全不变。

2.2 关键性质

性质 公式 含义
保长度 \(\|\mathbf{Qx}\|_2 = \|\mathbf{x}\|_2\) 不拉伸、不压缩
逆 = 转置 \(\mathbf{Q}^{-1} = \mathbf{Q}^\top\) 求逆零成本
条件数 = 1 \(\kappa_2(\mathbf{Q}) = 1\) 数值最稳定
乘积仍正交 \(\mathbf{Q}_1\mathbf{Q}_2\) 正交 旋转叠旋转还是旋转

3 矩阵的迹（Trace）

3.1 定义

\(n \times n\) 方阵 \(\mathbf{A}\) 的迹就是对角线元素之和：

\[ \text{tr}(\mathbf{A}) = \sum_{i=1}^{n} a_{ii} \]

只对方阵有定义，结果是一个标量。

              3, 4), nrow = 2, byrow = TRUE)       # byrow 参数指定按行生成矩阵
A
     [,1] [,2]
[1,]    1    2
[2,]    3    4
# 根据迹的定义，计算迹
sum(diag(A))  # 1 + 4 = 5
[1] 5
# R 没有内置 tr()，但可以自己写一个方便的辅助函数，计算矩阵的迹
tr <- function(A) {
    sum(diag(A))
}

3.2 基本性质

对 \(\mathbf{A}, \mathbf{B} \in \mathbb{R}^{n \times n}\)，\(c \in \mathbb{R}\)：

# 性质 公式
1 线性 \(\text{tr}(\mathbf{A}+\mathbf{B}) = \text{tr}(\mathbf{A})+\text{tr}(\mathbf{B})\)
2 齐次 \(\text{tr}(c\mathbf{A}) = c\,\text{tr}(\mathbf{A})\)
3 循环 \(\text{tr}(\mathbf{AB}) = \text{tr}(\mathbf{BA})\)
4 相似不变 \(\text{tr}(\mathbf{PAP}^{-1}) = \text{tr}(\mathbf{A})\)
5 转置不变 \(\text{tr}(\mathbf{A}^\top) = \text{tr}(\mathbf{A})\)

最关键的是循环性——三个矩阵也行：\(\text{tr}(\mathbf{ABC}) = \text{tr}(\mathbf{BCA}) = \text{tr}(\mathbf{CAB})\)。

但 \(\text{tr}(\mathbf{AB}) \neq \text{tr}(\mathbf{A})\cdot\text{tr}(\mathbf{B})\)，这是个常见误区。

3.3 迹 = 特征值之和

这是一个深刻的结论：计算特征值需要求解特征多项式，算法复杂度 \(O(n^3)\)，但是 迹 可以 \(O(n)\) 的复杂度下直接读取

若 \(\mathbf{B} = \mathbf{P}^{-1}\mathbf{A}\mathbf{P}\)，则：\(\text{tr}(\mathbf{B}) = \text{tr}(\mathbf{A})\)

  • 快速计算特征值之和
  • 验证特征值计算的正确性

\[ \text{tr}(\mathbf{A}) = \sum_{i=1}^{n} \lambda_i \]

  • 不是所有方阵都可以对角化，但是实对称矩阵，例如 \(A^TA\) 可以快速构建实对称矩阵，一定可以对角化
  • 对角化后，有 n 个不同的特征值
  • 但是，只要是方阵，都有特征值，但未必可以对角化。对角化的充要条件是有 \(n\) 个线性无关的特征向量；充分条件是有 \(n\) 个不同的特征值（能对角化不一定有 n 个不同特征值）

证明思路：如果 \(\mathbf{A} = \mathbf{P}\mathbf{D}\mathbf{P}^{-1}\)（对角化），那么由相似不变性 \(\text{tr}(\mathbf{A}) = \text{tr}(\mathbf{D}) = \sum \lambda_i\)。

用特征多项式证明：

\(\det(\mathbf{A} - \lambda\mathbf{I}) = (-1)^n(\lambda - \lambda_1)(\lambda - \lambda_2)\cdots(\lambda - \lambda_n)\)

展开后：

\(= (-1)^n\left[\lambda^n - (\lambda_1 + \cdots + \lambda_n)\lambda^{n-1} + \cdots\right]\)

另一方面，直接展开 \(\det(\mathbf{A} - \lambda\mathbf{I})\)，\(\lambda^{n-1}\) 项的系数是 \((-1)^{n-1}(a_{11} + a_{22} + \cdots + a_{nn})\)

比较系数得：

\(\lambda_1 + \lambda_2 + \cdots + \lambda_n = a_{11} + a_{22} + \cdots + a_{nn} = \text{tr}(\mathbf{A})\)

eig <- eigen(A)      # 特征分解
eig$values           # 特征值（可以是复数）
[1]  5.3722813 -0.3722813
sum(eig$values)      # 特征值之和
[1] 5
tr(A)                # 迹
[1] 5
# 两者相等

4 矩阵范数的一般定义

在介绍算子范数之前，先明确”矩阵范数”这个概念的公理。

函数 \(\|\cdot\| : \mathbb{R}^{m \times n} \to [0,\infty)\) 是矩阵范数，当且仅当对所有 \(\mathbf{A},\mathbf{B}\) 和标量 \(c\)：

  1. 正定性：\(\|\mathbf{A}\| = 0 \iff \mathbf{A} = \mathbf{0}\)
  2. 绝对齐次性：\(\|c\mathbf{A}\| = |c|\cdot\|\mathbf{A}\|\)
  3. 三角不等式：\(\|\mathbf{A}+\mathbf{B}\| \le \|\mathbf{A}\|+\|\mathbf{B}\|\)

这和向量范数的公理一模一样——事实上，向量范数就是 \(\mathbb{R}^{n \times 1}\) 上的矩阵范数。

6.1 定义

把矩阵所有元素平方加起来再开根号——就像把矩阵当成一个长向量，算欧几里得长度：

\[ \|\mathbf{A}\|_F = \sqrt{\sum_{i=1}^{m}\sum_{j=1}^{n} a_{ij}^2} \]

6.3 Frobenius 范数的问题/局限性

它把矩阵当成长向量来处理，忽略了矩阵作为线性变换的含义。下面的三个矩阵有相同的 Frobenius 范数，但做的变换完全不同：

A1 <- diag(2)                         # 恒等变换
A2 <- matrix(c(0,1,1,0), 2)          # 交换坐标
A3 <- matrix(c(sqrt(2),0,0,0), 2)    # 把第一分量拉伸，第二分量压到 0

cat("Frobenius 范数都是", norm(A1, "F"), "\n")
Frobenius 范数都是 1.414214 
# 但对 x = (1, 0) 的作用完全不同
x <- c(1, 0)
cat("A1·x =", A1 %*% x, " （不变）\n")
A1·x = 1 0  （不变）
cat("A2·x =", A2 %*% x, " （交换）\n")
A2·x = 0 1  （交换）
cat("A3·x =", A3 %*% x, " （拉伸+投影）\n")
A3·x = 1.414214 0  （拉伸+投影）

7 算子范数：衡量矩阵的拉伸能力

7.1 定义

给定一种向量范数 \(\|\cdot\|_V\)，矩阵 \(\mathbf{A}\) 的算子范数为：

\[ \|\mathbf{A}\|_V = \max_{\mathbf{x}\neq\mathbf{0}} \frac{\|\mathbf{A}\mathbf{x}\|_V}{\|\mathbf{x}\|_V} = \max_{\|\mathbf{x}\|_V=1} \|\mathbf{A}\mathbf{x}\|_V \]

两个写法等价，因为比值对 \(\mathbf{x}\) 的缩放不变——类比标量：\(\frac{|cx|}{|x|} = |c|\) 对所有非零 \(x\) 相同。

直观理解：拿所有长度为 1 的向量去乘 \(\mathbf{A}\)，看变换后最长能变成多少。这个最大长度就是算子范数。

7.2 可视化：单位圆变椭圆

8 谱范数与谱分解

8.1 核心定理

对于 \(\mathbf{A} \in \mathbb{R}^{m \times n}\)，秩为 \(r\)，谱分解 \(\mathbf{A}^\top\mathbf{A} = \mathbf{P}\boldsymbol{\Lambda}\mathbf{P}^\top\)，其中特征值 \(\lambda_1 \ge \lambda_2 \ge \cdots \ge \lambda_r > 0\)：

\[ \|\mathbf{A}\|_2 = \sqrt{\lambda_1} = \sqrt{\lambda_{\max}(\mathbf{A}^\top\mathbf{A})} \]

你不需要一个一个向量去试——直接算 \(\mathbf{A}^\top\mathbf{A}\) 的最大特征值然后开根号就行。

8.1.1 证明思路

\[ \|\mathbf{Ax}\|_2^2 = \mathbf{x}^\top\mathbf{A}^\top\mathbf{A}\mathbf{x} = \mathbf{x}^\top\mathbf{P}\boldsymbol{\Lambda}\mathbf{P}^\top\mathbf{x} \]

令 \(\mathbf{y} = \mathbf{P}^\top\mathbf{x}\)。因为 \(\mathbf{P}\) 是正交矩阵，\(\|\mathbf{x}\|_2 = 1\) 时 \(\|\mathbf{y}\|_2 = 1\)。所以：

\[ \max_{\|\mathbf{x}\|=1}\|\mathbf{Ax}\|_2^2 = \max_{\|\mathbf{y}\|=1} \mathbf{y}^\top\boldsymbol{\Lambda}\mathbf{y} = \max_{\|\mathbf{y}\|=1} \sum_i \lambda_i y_i^2 \le \lambda_1 \sum_i y_i^2 = \lambda_1 \]

当 \(\mathbf{y} = \mathbf{e}_1\)（第一个标准基向量）时取等号。

8.2 谱分解能干什么？

谱分解把矩阵拆开，让你看清每个方向上分别拉伸多少。拆开之后你能：

  • 算条件数：知道问题有多病态
  • 算范数：知道最大拉伸能力
  • 算矩阵的逆：对角矩阵求逆很简单
  • 算矩阵的幂：\(\mathbf{A}^{100}\) 轻松搞定
  • 降维：扔掉不重要的方向

谱分解、特征值分解、特征分解，都是同一个东西的不同叫法

  • A的F范数的平方就是\(A^TA\)的迹
  • A的F范数的平方就是\(A^TA\)的特征值之和
  • A的奇异值就是 \(A^TA\) 的特征值开根号
  • A的谱范数 就是 最大的奇异值
  • A的谱分解就是实对称矩阵的对角化，且特征向量正交
  • 如果A是实对称矩阵，eigen() 求的是特征值和特征向量，这就是谱分解的全部材料
  • 条件数就是最大奇异值除以最小奇异值
  1. 矩阵 \(\mathbf{A}\)
  2. → 计算实对称矩阵 \(\mathbf{A}^\top\mathbf{A}\)
  3. → 谱分解得到特征值 \(\lambda_i\) 和正交特征向量矩阵 \(\mathbf{Q}\)
  4. → 特征值构成对角矩阵 \(\mathbf{D} = \text{diag}(\lambda_1, \ldots, \lambda_n)\)（对角化）
  5. → 特征值开根号得到奇异值 \(\sigma_i = \sqrt{\lambda_i}\)
  6. → 最大奇异值就是谱范数 \(\|\mathbf{A}\|_2 = \sigma_{\max}\)

9 条件数：问题有多”病态”？

9.1 定义

\[ \kappa(\mathbf{A}) = \|\mathbf{A}\| \cdot \|\mathbf{A}^{-1}\| \]

用谱范数来算：\(\|\mathbf{A}\|_2 = \sigma_{\max}\)，\(\|\mathbf{A}^{-1}\|_2 = 1/\sigma_{\min}\)，所以

注意：根据奇异值分解定理，\(A^{−1}\)的奇异值是 A 的奇异值的倒数，所以，\(A^{−1}\)谱范数为A的最小范数的倒数。

\[ \kappa_2(\mathbf{A}) = \frac{\sigma_{\max}}{\sigma_{\min}} \]

条件数 含义 比喻
\(\kappa \approx 1\) 良态 圆碗——看错1cm，偏1cm
\(\kappa \gg 1\) 病态 窄峡谷——看错1cm，偏50cm
\(\kappa = \infty\) 奇异 矩阵不可逆

9.2 直观感受：圆碗 vs 峡谷

  • 奇异值代表矩阵在各方向的拉伸/压缩程度 ✓
  • 条件数 = 最大奇异值/最小奇异值 ✓
  • 条件数越大，椭圆越扁 ✓
  • 基于以上认知，求解方程组 Ax = b
  • x = \(A^{-1}b\) 是我们要求的解，本质上是针对输出结果b的逆变换
  • 即知道了椭圆的点，反推圆上的点
  • 因为b是有测量/系统带来的误差
  • 所以实际上我们观测到的是 b + Δb，而不是真正的 b
  • 我们求解的是 x = A⁻¹(b + Δb)，而不是真正的 x = A⁻¹b
  • 误差 Δb 也会被 A⁻¹ 变换，变成 Δx = A⁻¹Δb
  • 回到椭圆和圆的图像：椭圆上的小偏移 Δb，反推回圆上变成 Δx
  • 椭圆越扁，短轴方向上的小偏移，反推回去就越大
  • 所以条件数越大，b 的小误差可能导致 x 的大误差

9.4 条件数与 AI 训练

这不是抽象概念——它直接影响你训练神经网络的速度。

梯度下降在条件数大的”峡谷”里会来回弹跳，训练变慢甚至不收敛。实际中的应对策略：

  • 调学习率：步子迈小，弹跳减少，但太小就走得太慢
  • Adam 优化器：记住每个方向的历史，自动调整步长
  • Batch Normalization：让每一层的输入分布更均匀，减小条件数
  • 数据归一化：让输入特征量级一致，一开始就把峡谷压圆
# 模拟二维梯度下降：良态 vs 病态
gradient_descent <- function(A, b, lr, n_steps) {
  x <- c(0, 0)
  path <- matrix(NA, n_steps + 1, 2)
  path[1, ] <- x
  for (i in seq_len(n_steps)) {
    grad <- crossprod(A, A %*% x - b)  # A'(Ax - b)
    x <- x - lr * grad
    path[i + 1, ] <- x
  }
  path
}

10 特征值 vs 奇异值 vs 谱范数

这两个概念容易混淆，用表格理清：

特征值 \(\lambda\) 奇异值 \(\sigma\)
适用范围 方阵 任何矩阵
定义 \(\mathbf{Av} = \lambda\mathbf{v}\) \(\sigma_i = \sqrt{\lambda_i(\mathbf{A}^\top\mathbf{A})}\)
取值范围 实数或复数，可为负 非负实数
几何含义 特征方向上的拉伸倍数（含方向） 各方向的拉伸倍数（绝对值）

对于对称正定矩阵，两者相等。

正定矩阵的定义：

  • 对于任意非零向量 x，都有 xᵀAx > 0
  • 正定矩阵必须是对称矩阵

直观理解：

把 xᵀAx 想象成一个”能量”或”碗的形状”：

  • 正定矩阵：碗口朝上，有唯一最低点
  • 负定矩阵：碗口朝下，有唯一最高点
  • 不定矩阵：马鞍形，有的方向上升有的方向下降

等价条件：

  • 所有特征值 > 0
  • 所有奇异值 > 0（对于对称矩阵）
  • 矩阵可以做 Cholesky 分解

11 Schatten 范数：统一的框架

11.1 动机

算子范数只看最极端的那个方向。有时候你想知道整体情况：矩阵在所有方向上的拉伸加起来有多厉害？

Schatten 范数把每个方向的拉伸倍数（奇异值 \(\sigma_i\)）收集起来，用 \(p\)-范数统一度量：

\[ \|\mathbf{A}\|_{S,p} = \left(\sum_{i=1}^{r} \sigma_i^p\right)^{1/p} \]

11.2 三个重要范数用奇异值表示

\(p\) 名称 公式 用途
\(\infty\) 谱范数 \(\sigma_{\max}\) 最大拉伸
\(2\) Frobenius 范数 \(\sqrt{\sum \sigma_i^2}\) 整体大小
\(1\) 核范数 \(\sum \sigma_i\) 逼迫低秩

惊喜：Frobenius 范数一开始像是把矩阵当成长向量的粗暴算法，结果恰好等于 Schatten-2 范数——看似无脑的定义，背后藏着深刻的结构。

schatten <- function(A, p) {
  s <- svd(A)$d
  if (p == Inf) max(s) else sum(s^p)^(1/p)
}

12 范数的重要性质

12.1 次可乘性

\[\|\mathbf{AB}\| \le \|\mathbf{A}\|\cdot\|\mathbf{B}\|\]

直觉：A 最多拉伸 3 倍，B 最多拉伸 5 倍，AB 最多拉伸 15 倍（通常更小，因为拉伸方向不一定对齐）。

为什么重要：分析迭代算法。如果 \(\|\mathbf{A}\| < 1\)，那 \(\|\mathbf{A}^n\| \le \|\mathbf{A}\|^n \to 0\)——算法收敛。

12.3 等价性

所有矩阵范数都是等价的——相差不超过一个常数倍：

\[c\|\mathbf{A}\|_a \le \|\mathbf{A}\|_b \le C\|\mathbf{A}\|_a\]

13 误差分析中的范数

13.1 在机器学习里面的背景

回想前一个章节，方程 Ax = b，其中：

  • x - 模型权重
  • b - 目标值（即标记的结果，类似观察到的样本房价）
  • A - 特征矩阵（一堆样本数据，面积，房间数量，楼层等特征）

因为收到的房屋销售样本数据里面面积，楼层等数据可能存在误差，所以实际上：

你以为求解的是：

Ax = b

你实际求解的是：

(A + ΔA)x = (b + Δb)

其中：

  • ΔA 是特征矩阵的误差（面积测量误差、楼层录入错误等）
  • Δb 是标签的误差（房价记录不准等）

后向误差分析告诉我们：

  1. 如果矩阵良态（条件数小）
  • 数据误差对模型参数 x 影响小
  • 模型稳定、可靠
  1. 如果矩阵病态（条件数大）
  • 数据的微小误差会导致 x 剧烈变化
  • 模型不稳定、不可靠

实际意义

  • 不需要追求完美数据
  • 不需要追求精确解
  • 只要保证算法引入的误差 < 数据本身的误差就够了
  • 条件数帮助你判断模型对数据噪声的敏感程度

目标：找 \(\Delta\mathbf{A}\) 使得 \((\mathbf{A} + \Delta\mathbf{A})\tilde{\mathbf{x}} = \mathbf{b}\)

  • 维度设定：
    • \(\mathbf{A}\)：\(m \times n\) 矩阵
    • \(\tilde{\mathbf{x}}\)：\(n \times 1\) 向量
    • \(\mathbf{b}\)：\(m \times 1\) 向量
    • \(\mathbf{r} = \mathbf{A}\tilde{\mathbf{x}} - \mathbf{b}\)：\(m \times 1\) 向量，即残差
    • \(\Delta\mathbf{A}\)：\(m \times n\) 矩阵
  • 展开：
    • \(\mathbf{A}\tilde{\mathbf{x}} + \Delta\mathbf{A} \cdot \tilde{\mathbf{x}} = \mathbf{b}\)
    • \(\Delta\mathbf{A} \cdot \tilde{\mathbf{x}} = \mathbf{b} - \mathbf{A}\tilde{\mathbf{x}} = -\mathbf{r}\)
  • 外积：两个向量的外积：\(\mathbf{u} \cdot \mathbf{v}^\top\)，其中 \(\mathbf{u}\) 是 \(m \times 1\)，\(\mathbf{v}\) 是 \(n \times 1\)，结果是 \(m \times n\) 矩阵
  • 分析：\(\Delta\mathbf{A}\) 是 \(m \times n\) 矩阵，有 \(m \times n\) 个未知数，但方程 \(\Delta\mathbf{A} \cdot \tilde{\mathbf{x}} = -\mathbf{r}\) 只有 \(m\) 个约束，所以解不唯一，有无穷多个 \(\Delta\mathbf{A}\) 满足条件
  • 归一化的外积：\(\dfrac{-\mathbf{r} \cdot \tilde{\mathbf{x}}^\top}{|\tilde{\mathbf{x}}|^2}\) 是其中一个解
  • 满足 \(\Delta\mathbf{A} \cdot \tilde{\mathbf{x}} = -\mathbf{r}\) ✓
  • 绝对扰动：\(|\Delta\mathbf{A}|_2 = \dfrac{|\mathbf{r}|_2}{|\tilde{\mathbf{x}}|_2}\) 表示 \(\Delta\mathbf{A}\) 本身的大小， 利用了公式：秩1矩阵的范数，对于外积 \(\mathbf{u} \cdot \mathbf{v}^\top\)，有：\(|\mathbf{u} \cdot \mathbf{v}^\top|_2 = |\mathbf{u}|_2 \cdot |\mathbf{v}|_2\)
    • 左边是 矩阵的谱范数，等于最大奇异值
    • 右边是 向量的2-范数，即欧几里得范数: \(|\mathbf{u}|_2 = \sqrt{u_1^2 + u_2^2 + \cdots + u_m^2}\)
  • 相对扰动：\(\eta = \dfrac{|\Delta\mathbf{A}|_2}{|\mathbf{A}|_2} = \dfrac{|\mathbf{r}|_2}{|\mathbf{A}|_2 \cdot |\tilde{\mathbf{x}}|_2}\), 这个 \(\eta\) 就叫后向误差（backward error）
    • \(\eta \approx \epsilon_{\text{machine}}\)（如 \(10^{-16}\)）→ 算法后向稳定
    • \(\eta \gg \epsilon_{\text{machine}}\) → 算法不稳定
  • 意义：这个比值越小，说明：
    • \(\tilde{\mathbf{x}}\) 越接近真实解
    • 算法越稳定（后向稳定）
    • 只需微小扰动 \(\mathbf{A}\)，\(\tilde{\mathbf{x}}\) 就是精确解

13.2 后向误差分析

场景：你想求解 \(\mathbf{Ax} = \mathbf{b}\)，但计算机给出的是近似解 \(\tilde{\mathbf{x}}\)。

问题：这个 \(\tilde{\mathbf{x}}\) 到底是”哪个问题”的精确解？

答案：它精确地解决了扰动问题 \((\mathbf{A}+\Delta\mathbf{A})\tilde{\mathbf{x}} = \mathbf{b}\)，其中

\[ \Delta\mathbf{A} = -\frac{\mathbf{r}\tilde{\mathbf{x}}^\top}{\|\tilde{\mathbf{x}}\|^2}, \qquad \mathbf{r} = \mathbf{A}\tilde{\mathbf{x}} - \mathbf{b} \]

  • r 就是残差 ： 把近似解 x̃ 代入原方程，得到的”差距”。如果 x̃ 是精确解，r = 0

相对扰动：\(\dfrac{\|\Delta\mathbf{A}\|_2}{\|\mathbf{A}\|_2} = \dfrac{\|\mathbf{r}\|_2}{\|\mathbf{A}\|_2\cdot\|\tilde{\mathbf{x}}\|_2}\)

# 用坐标下降实现简单的 LASSO
soft_threshold <- function(z, gamma) sign(z) * pmax(abs(z) - gamma, 0)

lasso_cd <- function(X, y, lambda, n_iter = 200) {
  n <- nrow(X); p <- ncol(X)
  beta <- rep(0, p)
  for (iter in seq_len(n_iter)) {
    for (j in seq_len(p)) {
      r_j <- y - X[, -j, drop = FALSE] %*% beta[-j]
      z_j <- crossprod(X[, j], r_j) / n
      beta[j] <- soft_threshold(z_j, lambda / 2) / (crossprod(X[, j]) / n)
    }
  }
  beta
}

17 总结

17.1 知识地图

矩阵范数
├── 逐元素范数（把矩阵当向量）
│   ├── Frobenius 范数 ─── 也是 Schatten-2 ！
│   ├── 和范数
│   └── 最大范数（⚠️ 不次可乘）
├── 算子范数（衡量拉伸能力）
│   ├── 谱范数 (L₂) ─── = 最大奇异值 ─── 也是 Schatten-∞
│   ├── 列和范数 (L₁)
│   └── 行和范数 (L∞)
└── Schatten 范数（基于奇异值）
    ├── p=∞ → 谱范数
    ├── p=2  → Frobenius
    └── p=1  → 核范数（逼迫低秩）

17.2 核心公式卡

概念 公式
迹 \(\text{tr}(\mathbf{A}) = \sum a_{ii} = \sum \lambda_i\)
循环性 \(\text{tr}(\mathbf{AB}) = \text{tr}(\mathbf{BA})\)
Frobenius 与迹 \(\|\mathbf{A}\|_F^2 = \text{tr}(\mathbf{A}^\top\mathbf{A})\)
谱范数 \(\|\mathbf{A}\|_2 = \sigma_{\max} = \sqrt{\lambda_{\max}(\mathbf{A}^\top\mathbf{A})}\)
条件数 \(\kappa_2(\mathbf{A}) = \sigma_{\max}/\sigma_{\min}\)
次可乘性 \(\|\mathbf{AB}\| \le \|\mathbf{A}\|\cdot\|\mathbf{B}\|\)
相容性 \(\|\mathbf{Ax}\| \le \|\mathbf{A}\|\cdot\|\mathbf{x}\|\)

17.3 什么时候用哪个范数？

你想做什么 用什么范数
衡量矩阵整体大小 Frobenius
衡量最大拉伸能力 谱范数
逼迫矩阵低秩 核范数
快速粗算 列和 / 行和范数
分析误差放大 条件数（= 谱范数的比）

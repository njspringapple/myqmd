1.2 KL 散度的定义

Definition der KL-Divergenz

我们需要一把”尺子”来衡量两个分布 \(G(\cdot)\) 和 \(F(\cdot)\) 之间的差异。这把尺子就是 Kullback-Leibler 散度（KL-Divergenz）。

Important定义 3.1：KL 散度 | Definition 3.1: KL-Divergenz

对于两个分布函数 \(G(\cdot)\) 和 \(F(\cdot)\)，其密度或概率质量函数分别为 \(g(y)\) 和 \(f(y)\)，KL 散度定义为：

\[ KL(G, F) = \int \log \frac{g(y)}{f(y)} \, dG(y) \]

若分布是连续的：

\[ KL(G, F) = \int \log \frac{g(y)}{f(y)} \, g(y) \, dy = \underbrace{\int \log g(y) \cdot g(y) \, dy}_{\text{负熵 | negative Entropie}} - \underbrace{\int \log f(y) \cdot g(y) \, dy}_{\text{交叉熵 | Kreuzentropie}} \]

若分布是离散的，积分替换为求和。

直觉：KL 散度衡量的是”用 \(F\) 去近似 \(G\) 时，你会有多’意外’“。如果 \(F\) 和 \(G\) 完全一致，你永远不会意外，KL 为 0。如果 \(F\) 在某些地方与 \(G\) 差距很大，你就会很”意外”，KL 就大。

其中 \(\int \log g(y) \cdot g(y) \, dy\) 称为熵（Entropie），对于离散随机变量 \(Y\)，熵是正的，除非分布退化为在某个点 \(y_0\) 处的单点质量。

1.3 KL 散度的性质

Eigenschaften der KL-Divergenz

1.3.1 性质 1：KL 散度始终非负 | KL-Divergenz ist immer nicht-negativ

\[ KL(G, F) \geq 0, \quad \text{等号成立当且仅当 } G(\cdot) \equiv F(\cdot) \]

Tip点击展开证明 | Klicken zum Aufklappen des Beweises

利用不等式 \(\log(y) \leq y - 1\)（等号在 \(y = 1\) 时成立）：

\[ \begin{aligned} KL(G, F) &= \int \log \frac{g(y)}{f(y)} g(y) \, dy \\ &= -\int \log \frac{f(y)}{g(y)} g(y) \, dy \\ &\geq \int \left(1 - \frac{f(y)}{g(y)}\right) g(y) \, dy \\ &= \int g(y) \, dy - \int f(y) \, dy \\ &= 1 - 1 = 0. \end{aligned} \]

1.3.2 性质 2：KL 散度不对称 | KL-Divergenz ist nicht symmetrisch

\[ KL(G, F) \neq KL(F, G) \quad \text{(通常情况下 | im Allgemeinen)} \]

因此 KL 散度不是度量（Metrik），因为度量要求对称性。但我们可以固定 \(G(\cdot)\) 作为”锚点”来使用它。

# 两个正态分布的 KL 散度解析公式
# Analytische Formel der KL-Divergenz fur zwei Normalverteilungen
kl_normal <- function(mu1, sigma1, mu2, sigma2) {
  log(sigma2 / sigma1) + (sigma1^2 + (mu1 - mu2)^2) / (2 * sigma2^2) - 0.5
}

kl_gf <- kl_normal(0, 1, 0.8, 1.3)
kl_fg <- kl_normal(0.8, 1.3, 0, 1)

1.4 从 KL 散度到最优参数 \(\theta_0\)

Von der KL-Divergenz zum optimalen Parameter \(\theta_0\)

模型族 \(\mathcal{F}\) 中离 \(G(\cdot)\) 最近的分布，由最小化 KL 散度得到：

\[ KL(G(\cdot), \mathcal{F}) := \min_{F(\cdot;\theta) \in \mathcal{F}} KL(G(\cdot), F(\cdot;\theta)) \tag{3.2} \]

假设 \(\mathcal{F}\) 是 Fisher 正则的（可微、积分与微分可交换），令导数为零：

\[ 0 = \frac{\partial}{\partial \theta} \int \log \frac{g(y)}{f(y;\theta)} \, dG(y) = -\int \frac{\partial}{\partial \theta} \log f(y;\theta) \, dG(y) \]

定义最优参数 \(\theta_0\)：

\[ \boxed{0 = \int \frac{\partial \log f(y;\theta_0)}{\partial \theta} \, dG(y) = E\!\left(\frac{\partial \log f(Y;\theta_0)}{\partial \theta}\right)} \tag{3.3} \]

\(\theta_0\) 被称为”真参数”——它是在你的画法体系内，理论上最接近真实地形的参数。

Important关键洞察 | Schlusselerkenntnis

\(\theta_0\) 无法在实际中直接计算，因为积分 \(\int \cdot \, dG(y)\) 需要知道 \(G(\cdot)\)，而 \(G(\cdot)\) 正是我们不知道的。但大数定律告诉我们，对于任何可测函数 \(t(Y)\)：

\[ \frac{1}{n}\sum_{i=1}^{n} t(y_i) \xrightarrow{n \to \infty} E\big(t(Y)\big) = \int t(y) \, dG(y) \]

这就是从”理论最优”走向”实际可算”的桥梁。

2.1 从 KL 散度到似然：一步之遥

Von der KL-Divergenz zur Likelihood: Nur ein Schritt

回顾 KL 散度的分解：

\[ KL(G, F(\cdot;\theta)) = \underbrace{\int \log g(y) \cdot g(y) \, dy}_{\text{G 的负熵（不依赖 } \theta \text{）}} - \underbrace{\int \log f(y;\theta) \cdot g(y) \, dy}_{\text{交叉熵（依赖 } \theta \text{）}} \]

第一项是 \(G\) 的负熵，不含 \(\theta\)，无法估计也不需要估计（因为我们不知道 \(g(\cdot)\)，无法计算 \(\frac{1}{n}\sum \log g(y_i)\)）。第二项可以用样本平均估计：

\[ \frac{1}{n} \sum_{i=1}^{n} \log f(y_i; \theta) \xrightarrow{n \to \infty} \int \log f(y;\theta) \cdot g(y) \, dy \]

最小化 KL 散度 = 最大化第二项 = 最大化对数似然！

Minimierung der KL-Divergenz = Maximierung der Log-Likelihood!

2.2 对数似然函数

Die Log-Likelihood-Funktion

Important定义 3.2：对数似然函数 | Definition 3.2: Log-Likelihood-Funktion

\[ l(\theta) = \sum_{i=1}^{n} \log f(y_i; \theta) \]

其中 \(y_1, \ldots, y_n\) 是观测数据，\(f(\cdot;\theta)\) 是模型的密度函数。

似然函数依赖于观测 \(y_1,\ldots,y_n\)，记号上常省略不写。严格地说 \(l(\theta) = l(\theta; y_1, \ldots, y_n)\)。

似然函数的直觉：把它想象成一个侦探的评分系统。现场留下了证据 \((y_1, \ldots, y_n)\)，每个嫌疑人对应一个参数值 \(\theta\)。似然函数问的是：“如果嫌疑人就是 \(\theta\)，留下这些证据的可能性有多大？”评分最高的嫌疑人就是最大似然估计（MLE）。

Likelihood = Bewertungssystem eines Detektivs. Welcher Verdachtige (\(\theta\)) erklart die Beweise (\(y_1,\ldots,y_n\)) am besten?

注意 \(\theta_0\) 由 \(\int \frac{\partial \log f(y;\theta_0)}{\partial \theta} dG(y) = 0\) 定义。我们用其经验版本替换：

\[ \boxed{0 = \sum_{i=1}^{n} \frac{\partial \log f(y_i; \hat{\theta})}{\partial \theta}} \tag{3.4} \]

其中 \(\hat{\theta}\) 求解 (3.4)。我们定义 \(\hat{\theta}\) 为最大似然估计（Maximum-Likelihood-Schatzung, MLE）。

此时有个看似矛盾之处：“最大”似然明明源自”最小”KL 散度。这是因为最小化 KL 等价于最大化其中的负交叉熵项，而省略常数 \(\frac{1}{n}\) 后正好就是对数似然。

2.3 平方损失与正态分布

Quadratischer Verlust und Normalverteilung

假设 \(\mathcal{F}\) 是所有均值为 \(\mu\)、方差为 \(\sigma^2\) 的正态分布的集合。对数似然为：

\[ l(\mu, \sigma^2) = \sum_{i=1}^{n} \left(-\log(\sigma) - \frac{1}{2}\frac{(y_i - \mu)^2}{\sigma^2}\right) \tag{3.6} \]

如果我们只对 \(\mu\) 感兴趣（暂时忽略 \(\sigma^2\)），那么：

\[ \text{最大化 } l(\mu) \iff \text{最小化 } \sum_{i=1}^{n}(y_i - \mu)^2 \tag{3.5} \]

Important关键等价关系 | Schlusselaquivalenz

使用平方损失 \(\Longleftrightarrow\) 假设数据服从正态分布

这不是巧合，而是 KL 散度理论的直接推论：

\[ \text{最小二乘} = \text{正态分布下的最大似然估计} \]

Kleinste Quadrate = Maximum-Likelihood-Schatzung unter Normalverteilung.

3.1 概率的两种角色

Zwei Rollen der Wahrscheinlichkeit

在经典统计中，给定参数 \(\theta\)，我们构建变量 \(Y\) 的概率模型 \(F(\cdot;\theta)\)。贝叶斯方法则反过来：给定观测到的数据 \(y\)，我们问”\(\theta\) 的概率分布是什么？”

这就像侦探收集了证据后，评估每个嫌疑人的”嫌疑概率”。这种反转在历史上称为”逆概率（inverse Wahrscheinlichkeit）“。

3.2 先验分布

Die Prior-Verteilung

先验分布（Prior-Verteilung） \(p(\theta)\) 编码了我们在看到数据之前对 \(\theta\) 的信念。其结构可能包含额外的参数，暂时忽略。

  • 如果你对某片土地一无所知，先验是”平坦的”（均匀分布）
  • 如果你是老制图师，对地形有经验，先验就有”偏好”

3.3 后验分布

Die Posterior-Verteilung

通过贝叶斯公式，我们将先验信念与数据结合，得到后验分布（Posterior-Verteilung）：

\[ p(\theta | y) = \frac{f(y;\theta) \, p(\theta)}{f(y)} \overset{\text{i.i.d.}}{=} \frac{\prod_{i=1}^{n} f(y_i;\theta) \, p(\theta)}{f(y)} = \frac{\exp(l(\theta)) \, p(\theta)}{f(y)} \tag{3.7} \]

其中：

  • 分子第一项 \(\exp(l(\theta))\) 是似然（对数似然的指数）
  • \(p(\theta)\) 是先验密度
  • 归一化常数 \(f(y) = \int_{\theta \in \Theta} \prod_{i=1}^{n} f(y_i;\theta) \, p(\theta) \, d\theta\) 通常数值上难以计算（将在第 9 章详细讨论）

3.4 后验众数与 MLE 的关系

Posteriorer Modus und MLE

Important定义 3.3：后验众数 | Definition 3.3: Posteriorer Modus

\[ \hat{\theta}_{\text{postmode}} = \arg\max_{\theta \in \Theta} \, p(\theta | y) \]

关键联系：如果先验 \(p(\theta)\) 是常数（所谓”平坦先验 / flat prior”），则：

\[ \arg\max_{\theta \in \Theta} \, p(\theta | y) = \arg\max_{\theta \in \Theta} \, l(\theta) \tag{3.8} \]

即后验众数 = MLE。先验不平坦时，两者一般不同——先验会”拉”后验众数偏离 MLE。

4 估计量的性质

Eigenschaften von Schatzern

Note本节目标 | Was Sie lernen

  • 从 KL 收敛到 MSE 的完整推导，引出 Fisher 信息矩阵
  • 偏差-方差权衡 (Bias-Varianz-Tradeoff)：打靶的两种偏离
  • 充分性 (Suffizienz)：压缩数据而不丢失信息

4.1 相合性：更多数据，更好估计

Konsistenz: Mehr Daten, bessere Schatzung

直觉：GPS 接收到的卫星信号越多，定位越精确。类似地，估计量 \(\hat{\theta}\) 随样本量 \(n\) 增大而趋近真参数 \(\theta_0\)。

4.1.1 通过 KL 散度表述相合性

Konsistenz durch KL-Divergenz formulieren

相合性意味着随着数据量增大：

\[ \int \log \frac{f(y;\hat{\theta})}{f(y;\theta_0)} \, dG(y) \xrightarrow{n \to \infty} 0 \tag{3.9} \]

简单计算表明：

\[ \int \log \frac{f(y;\hat{\theta})}{f(y;\theta_0)} \, dG(y) = KL(G(\cdot), F(\cdot;\theta_0)) - KL(G(\cdot), F(\cdot;\hat{\theta})) \]

因此要求两个 KL 散度随 \(n\) 增大而趋于一致。

4.1.2 Taylor 展开与 Fisher 信息矩阵

Taylor-Entwicklung und Fisher-Informationsmatrix

对 \(\log f(y;\hat{\theta})\) 在 \(\theta_0\) 处做二阶 Taylor 展开：

\[ \log f(y;\hat{\theta}) \approx \log f(y;\theta_0) + \frac{\partial \log f(y;\theta_0)}{\partial \theta}(\hat{\theta} - \theta_0) + \frac{1}{2}(\hat{\theta}-\theta_0)^\top \frac{\partial^2 \log f(y;\theta_0)}{\partial\theta\partial\theta^\top}(\hat{\theta}-\theta_0) \]

代入 (3.9)，利用 (3.3) 中 \(E\!\left[\frac{\partial \log f}{\partial\theta}\Big|_{\theta_0}\right]=0\)（一阶项对 \(G\) 积分后消失），得到：

\[ 2\int \log\frac{f(y;\hat\theta)}{f(y;\theta_0)}\,dG(y) \approx (\hat\theta-\theta_0)^\top \underbrace{E\!\left(-\frac{\partial^2 \log f(y;\theta_0)}{\partial\theta\partial\theta^\top}\right)}_{:=\;I(\theta_0)}\;(\hat\theta-\theta_0) \tag{3.10} \]

其中期望关于数据生成过程 \(G(\cdot)\) 取。\(I(\theta_0)\) 称为 Fisher 信息矩阵（Fisher-Informationsmatrix），它度量了对数似然曲面在 \(\theta_0\) 处的”曲率”——曲率越大，数据对参数越”敏感”，估计越精确。Fisher 信息在最大似然估计中起着核心作用。

由于 \(\hat{\theta} = t(y_1,\ldots,y_n)\) 是数据的函数，而数据是随机的，因此 \(\hat{\theta}\) 也是随机的。对所有可能的样本取期望：

\[ E\!\left(\int \log \frac{f(y;\hat{\theta})}{f(y;\theta_0)} \, dG(y)\right) \xrightarrow{n\to\infty} 0 \]

其中期望关于数据 \(Y_1,\ldots,Y_n\) 取。形式上，上式等于：

\[ \underbrace{\int \cdots \int}_{n \text{ 重积分}} \int \log \frac{f(y; t(y_1,\ldots,y_n))}{f(y;\theta_0)} \, dG(y)\,dG(y_1)\cdots G(y_n) \]

利用 Taylor 近似 (3.10)，简化为：

\[ E\!\Big[(\hat\theta-\theta_0)^\top I(\theta_0)(\hat\theta-\theta_0)\Big] \xrightarrow{n\to\infty} 0 \]

对单参数情形（\(I(\theta_0)>0\) 为常数），可提出为：

\[ E\!\Big[I(\theta_0)(\hat\theta-\theta_0)^2\Big] \propto E\!\Big[(\hat\theta-\theta_0)^2\Big] \]

4.1.3 偏差-方差分解

Bias-Varianz-Zerlegung

将上式展开（添加减 \(E(\hat\theta)\)）：

\[ E\!\big[(\hat\theta-\theta_0)^2\big] = E\!\big[(\hat\theta - E(\hat\theta) + E(\hat\theta) - \theta_0)^2\big] = \underbrace{E\!\big[(\hat\theta-E(\hat\theta))^2\big]}_{Var(\hat\theta)} + \underbrace{(E(\hat\theta)-\theta_0)^2}_{bias^2(\hat\theta,\theta_0)} \tag{3.11} \]

Important定义 3.4：均方误差 | Definition 3.4: Mittlerer Quadratischer Fehler (MSE)

\[ MSE(\hat{\theta}, \theta_0) = E\!\left[(\hat{\theta} - \theta_0)^2\right] = Var(\hat{\theta}) + bias^2(\hat{\theta}, \theta_0) \]

偏差定义为 \(bias(\hat{\theta}, \theta_0) = E(\hat{\theta}) - \theta_0\)。

  • 若 \(bias = 0\)，称估计量无偏（unverzerrt）
  • 若 \(bias \xrightarrow{n\to\infty} 0\)，称为渐近无偏（asymptotisch unverzerrt）

4.1.4 MSE 相合性与概率相合性

MSE-Konsistenz und Konsistenz in Wahrscheinlichkeit

Important定义 3.5：MSE 相合性 | Definition 3.5: MSE-Konsistenz

估计量 \(\hat{\theta}\) 对 \(\theta_0\) 是 MSE 相合的，如果：

\[ MSE(\hat{\theta}, \theta_0) \xrightarrow{n \to \infty} 0 \]

另一种等价定义：对任意 \(\delta > 0\)，

\[ P(|\hat{\theta} - \theta_0| \geq \delta) \xrightarrow{n \to \infty} 0 \tag{3.12} \]

利用切比雪夫（Chebyshev）不等式——对任何均值为 \(\mu\)、方差为 \(\sigma^2 > 0\) 的随机变量 \(X\)：

\[ P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2} \]

取 \(\hat{\theta}\) 为随机变量，得到：

\[ P(|\hat{\theta}-\theta_0| \geq \delta) \leq \frac{E[(\hat{\theta}-\theta_0)^2]}{\delta^2} = \frac{MSE(\hat\theta,\theta_0)}{\delta^2} \xrightarrow{n\to\infty} 0 \]

因此 MSE 相合 \(\Leftrightarrow\) 概率相合（在适当的正则条件下）。

4.2 充分性：压缩数据，不丢信息

想象你要把一本500页的勘测报告寄给远方的同事，但邮费按重量收。充分统计量就是那份完美的摘要——它把关于参数 \(\theta\) 的所有信息浓缩在里面，原始报告可以扔掉。

例如，考虑 \(Y_i \sim \text{Binom}(1, \pi)\)，算术平均 \(\bar{Y} = \sum Y_i / n\) 是 \(\pi\) 的 MLE。对于 \(n=4\) 的以下四组样本：

  • \(y = (1,0,0,0)\), \(y = (0,1,0,0)\), \(y = (0,0,1,0)\), \(y = (0,0,0,1)\)

它们的 MLE 完全相同（\(\bar{y} = 1/4\)）。观测的具体顺序不提供关于 \(\pi\) 的任何信息。

Important定义 3.6：充分统计量 | Definition 3.6: Suffiziente Statistik

统计量 \(t(Y_1, \ldots, Y_n)\) 对参数 \(\theta\) 是充分的，如果给定 \(t\) 的值后，数据的条件分布不再依赖 \(\theta\)：

\[ P(Y_1 = y_1, \ldots, Y_n = y_n \mid t(Y_1, \ldots, Y_n) = t_0;\, \theta) \text{ 不依赖 } \theta \]

如果 \(t(y_1,\ldots,y_n)\) 给出了关于 \(\theta\) 的充分信息，那么知道 \(t\) 就足以估计参数——我们甚至可以删掉原始数据。在大数据时代，这意味着只需存储 \(t\) 就够了。

Tip点击展开：伯努利充分性的完整证明 | Vollstandiger Beweis fur Bernoulli-Suffizienz

4.2.1 Neyman 分解定理 | Neyman-Faktorisierungssatz

直接用定义验证充分性有时很困难。Neyman 分解提供了一个简便的判据。

Important性质 3.1：Neyman 分解 | Eigenschaft 3.1: Neyman-Faktorisierung

统计量 \(t(y_1, \ldots, y_n)\) 对 \(\theta\) 充分，当且仅当密度函数可以分解为：

\[ f(y_1, \ldots, y_n; \theta) = h(y_1, \ldots, y_n) \cdot g\!\big(t(y_1, \ldots, y_n);\, \theta\big) \tag{3.13} \]

其中 \(h(\cdot)\) 不依赖 \(\theta\)，\(g(\cdot)\) 只通过统计量 \(t\) 依赖数据。

Tip点击展开：Neyman 分解的完整证明 | Vollstandiger Beweis der Neyman-Faktorisierung

(\(\Rightarrow\)) 设 \(t(\cdot)\) 充分。由条件概率的定义：

\[ f(y_1,\ldots,y_n;\theta) = \underbrace{f(y_1,\ldots,y_n \mid t(y)=t;\,\theta)}_{\text{不含}\;\theta\;\text{（充分性）}=:\,h(y_1,\ldots,y_n)} \cdot \underbrace{f_t(t \mid y_1,\ldots,y_n;\,\theta)}_{\text{只通过}\;t\;\text{依赖数据}=:\,g(t;\theta)} \]

(\(\Leftarrow\)) 设密度按 (3.13) 分解。\(t\) 的边际密度为：

\[ f_t(t;\theta) = \int_{t(y_1,\ldots,y_n)=t} f(y_1,\ldots,y_n;\theta)\,dy_1\cdots dy_n = g(t;\theta)\int_{t(y_1,\ldots,y_n)=t} h(y_1,\ldots,y_n)\,dy_1\cdots dy_n \]

条件分布为：

\[ f(y_1,\ldots,y_n \mid t=t;\,\theta) = \frac{f(y_1,\ldots,y_n,\,t;\,\theta)}{f_t(t;\theta)} = \begin{cases} \frac{h(y_1,\ldots,y_n)\,g(t;\theta)}{g(t;\theta)\int h\,dy} & \text{当 } t(y_1,\ldots,y_n)=t\\ 0 & \text{否则} \end{cases} \]

\(g(t;\theta)\) 消去，不依赖 \(\theta\)。\(\square\)

5.1 估计的随机性

Die Zufalligkeit der Schatzung

\(\hat{\theta}\) 不是一个固定数字——它是数据的函数，而数据是从 (3.1) 中随机抽取的。不同的数据集会给出不同的 \(\hat{\theta}\)。这种变异性称为估计方差（Schatz-Varianz），我们需要量化它。

由中心极限定理，在很多情况下：

\[ \hat{\theta} = t(Y_1, \ldots, Y_n) \overset{a}{\sim} N\!\left(\theta,\, Var(\hat{\theta})\right) \tag{3.15} \]

其中 \(\overset{a}{\sim}\) 表示”当 \(n\) 足够大时近似服从”。

5.2 置信区间

Konfidenzintervalle

与其给出一个点估计 \(\hat{\theta}\)，不如给出一个区间，使我们”有信心”认为真参数在其中。区间的上下界 \(t_l, t_r\) 都是数据的函数，因此区间本身是随机的。

Important定义 3.9：置信区间 | Definition 3.9: Konfidenzintervall

区间 \(CI = [t_l(Y_1,\ldots,Y_n),\; t_r(Y_1,\ldots,Y_n)]\) 是参数 \(\theta\) 的置信水平 \((1-\alpha)\) 的置信区间，如果对所有 \(\theta\)：

\[ P_\theta\!\Big(t_l(Y_1,\ldots,Y_n) \leq \theta \leq t_r(Y_1,\ldots,Y_n)\Big) \geq 1 - \alpha \tag{3.14} \]

注意 \(t_l = -\infty\), \(t_r = +\infty\) 总是满足定义（\(P(-\infty \leq \theta \leq \infty) = 1\)），但这样的区间毫无意义。目标是找到满足定义且长度最短的区间。实践中我们优先选择 (3.14) 中取等号。

5.2.1 枢轴量与构造方法

Pivotgrosse und Konstruktionsmethode

Important定义 3.10：枢轴量 | Definition 3.10: Pivotgrosse

量 \(g(t(Y_1,\ldots,Y_n);\, \theta)\) 称为枢轴量，如果它的分布不依赖 \(\theta\)。该分布称为枢轴分布。

精确的枢轴量是稀少的。但由于中心极限定理，近似枢轴分布很常见。利用 (3.15)：

\[ g = \frac{\hat{\theta} - \theta}{\sqrt{Var(\hat{\theta})}} \overset{a}{\sim} N(0, 1) \tag{3.16} \]

\(N(0,1)\) 不依赖 \(\theta\)，所以是枢轴的。由此构造置信区间：

\[ 1-\alpha \approx P\!\left(z_{\alpha/2} \leq \frac{\hat\theta-\theta}{\sqrt{Var(\hat\theta)}} \leq z_{1-\alpha/2}\right) \]

因为 \(z_{\alpha/2} = -z_{1-\alpha/2}\)，得到：

\[ \boxed{CI = \left[\hat{\theta} - z_{1-\alpha/2}\sqrt{Var(\hat{\theta})},\;\; \hat{\theta} + z_{1-\alpha/2}\sqrt{Var(\hat{\theta})}\right]} \tag{3.17} \]

当 \(1-\alpha = 0.95\) 时，\(z_{1-\alpha/2} = 1.96 \approx 2\)，给出常用近似：

\[ CI \approx \left[\hat{\theta} \pm 2\sqrt{Var(\hat{\theta})}\right] \tag{3.18} \]

5.3 正态均值的置信区间与 \(t\) 分布

Konfidenzintervall fur den Normalverteilungs-Mittelwert und die \(t\)-Verteilung

当数据 \(Y_i \sim N(\mu, \sigma^2)\) i.i.d.（式 3.19）时，\(\hat\mu = \bar{Y} \sim N(\mu, \sigma^2/n)\)。

若 \(\sigma^2\) 已知：

\[ \frac{\bar{Y} - \mu}{\sigma / \sqrt{n}} \sim N(0, 1) \quad \Rightarrow \quad CI = \left[\bar{y} - z_{1-\alpha/2}\frac{\sigma}{\sqrt{n}},\;\bar{y} + z_{1-\alpha/2}\frac{\sigma}{\sqrt{n}}\right] \tag{3.20} \]

但实际中 \(\sigma^2\) 未知，用 \(\hat\sigma^2 := S_n^2 = \frac{1}{n-1}\sum(Y_i - \bar{Y})^2\) 估计（\(n-1\) 使估计无偏，留作练习 1 验证）。此时：

\[ \frac{\bar{Y} - \mu}{\hat\sigma / \sqrt{n}} \not\sim N(0, 1) \tag{3.21} \]

William Sealy Gosset（1876-1937，笔名 “Student”） 在吉尼斯啤酒厂工作时发现并解决了这个问题。他因担心泄露商业秘密而以笔名发表（Student, 1908）。他的工作引出了 \(t\) 分布：

\[ \boxed{\frac{\bar{Y} - \mu}{S_n / \sqrt{n}} \sim t_{(n-1)}} \tag{3.22} \]

其中 \(t_{(n-1)}\) 是自由度为 \(n-1\) 的 \(t\) 分布。这同样是枢轴量，因为它既不依赖 \(\mu\) 也不依赖 \(\sigma^2\)。

Tip点击展开：\(t\) 分布的完整证明（教材式 3.22 / 3.27） | Vollstandiger Beweis der \(t\)-Verteilung

构造辅助变量：

\[ \begin{aligned} Z_1 &= \tfrac{1}{\sqrt{n}}(Y_1+Y_2+\cdots+Y_n) = \sqrt{n}\,\bar{Y}\\ Z_2 &= \tfrac{1}{\sqrt{2}}(Y_1-Y_2)\\ Z_3 &= \tfrac{1}{\sqrt{3}\sqrt{2}}(Y_1+Y_2-2Y_3)\\ &\;\;\vdots\\ Z_n &= \tfrac{1}{\sqrt{n}}\tfrac{1}{\sqrt{n-1}}\big(Y_1+\cdots+Y_{n-1}-(n-1)Y_n\big) \end{aligned} \]

性质（可逐一验证）：

  • 所有 \(Z_i\) 都是正态的（独立正态之和）
  • \(E(Z_1)=\sqrt{n}\mu\)，\(E(Z_i)=0\)（\(i\geq 2\)）
  • \(Var(Z_i)=\sigma^2\) 对所有 \(i\)
  • \(Cov(Z_i, Z_j)=0\) 对 \(i\neq j\)（详细计算见教材 p.38-39）

因此 \(Z_1,\ldots,Z_n\) 独立，且

\[ \frac{Z_1-\sqrt{n}\mu}{\sqrt{\sum_{i=2}^n Z_i^2/(n-1)}} \sim t_{(n-1)} \tag{3.27} \]

关键等式（用归纳法证明，见教材 p.39-40）：

\[ Z_2^2+\cdots+Z_n^2 = \sum_{i=1}^n(Y_i-\bar{Y})^2 = (n-1)S_n^2 \tag{3.28} \]

将 \(Z_1=\sqrt{n}\bar{Y}\) 和 (3.28) 代入 (3.27)，即得：

\[ T = \frac{\bar{Y}-\mu}{S_n/\sqrt{n}} = \frac{Z_1-\sqrt{n}\mu}{\sqrt{\sum_{i=2}^n Z_i^2/(n-1)}} \sim t_{(n-1)} \qquad \square \]

5.4 置信区间的正确解读

Korrekte Interpretation des Konfidenzintervalls

Warning注意：不要这样说 | Nicht so formulieren

“真参数有 95% 的概率落在 \([2.24, 4.16]\) 内。”

这是错误的。数据已经给定，\([2.24, 4.16]\) 是固定数字，\(\mu\) 也是固定数字——要么在里面，要么不在，没有概率可言。概率陈述 (3.24) 不是关于随机过程的。

Important定义 3.11：置信原理 | Definition 3.11: Konfidenzprinzip

如果我们用相同方法对相同大小的新数据重复计算，那么在 \(\alpha \times 100\%\) 的情况下，真参数不在区间内；在 \((1-\alpha) \times 100\%\) 的情况下，真参数在区间内。

这描述的是方法的长期频率性质，而不是某个特定区间的概率。

不过从不确定性的视角（回忆第 2 章掷骰子杯未揭开的例子），数据已给定，我们可以利用概率来量化对 \(\theta\) 的不确定性。我们可以说：以 \((1-\alpha)\times 100\%\) 的置信度相信参数在区间内——但这里的”概率”不是指真正的随机实验，而是对知识的量化。

5.5 可信区间：贝叶斯视角

Credibility Intervall: Die Bayessche Perspektive

通过贝叶斯推理，可以给出类似的区间解读。在贝叶斯统计中，\(\theta\) 被视为随机的，我们可以直接查看后验分布。

Important定义 3.12：可信区间 | Definition 3.12: Credibility Intervall

在贝叶斯框架中，\(\theta\) 是随机变量。可信区间 \([t_l(y), t_r(y)]\) 定义为：

\[ P_\theta\!\Big(\theta \in [t_l(y_1,\ldots,y_n),\, t_r(y_1,\ldots,y_n)] \;\Big|\; y_1,\ldots,y_n\Big) \geq 1 - \alpha \]

一种自然选择是等尾可信区间（两侧各切掉 \(\alpha/2\) 的概率）：

\[ \int_{-\infty}^{t_l} p_\theta(\theta | y) \, d\theta = \int_{t_r}^{\infty} p_\theta(\theta | y) \, d\theta = \frac{\alpha}{2} \tag{3.25} \]

5.5.1 最高后验密度区间（HDI）

等尾区间构造简单，但当后验分布不对称时，它会包含低密度区域而排除高密度区域。

HDI（Highest Density Interval） 解决了这个问题：选择一个阈值 \(c\)，使得所有密度大于 \(c\) 的点构成的区域恰好覆盖 \(1-\alpha\) 的概率。

\[ HDI(y_1, \ldots, y_n) = \{\theta : p_\theta(\theta | y_1, \ldots, y_n) \geq c\} \]

其中 \(c\) 满足 \(\int_{\theta \in HDI} p_\theta(\theta | y_1,\ldots,y_n) d\theta = 1-\alpha\)。

find_hdi <- function(a, b, prob = 0.9, n = 10000) {
  grid <- seq(0, 1, length.out = n)
  d <- dbeta(grid, a, b)
  sorted <- sort(d, decreasing = TRUE)
  cum <- cumsum(sorted) / sum(sorted)
  c_thresh <- sorted[which(cum >= prob)[1]]
  in_hdi <- d >= c_thresh
  c(min(grid[in_hdi]), max(grid[in_hdi]))

对于 i.i.d. 随机变量 \(Y_i\)，均值 \(\mu\)，方差 \(\sigma^2\)，证明

\[ S_n^2 = \frac{1}{n-1}\sum_{i=1}^n (Y_i - \bar{Y})^2 \]

是 \(\sigma^2\) 的无偏估计。提示：将被加数改写为 \((Y_i - \mu) - (\bar{Y} - \mu)\)。

# 数值验证 S_n^2 的无偏性
set.seed(42)
mu_ex <- 10; sigma_ex <- 3; n_ex <- 15

s2_samples <- replicate(50000, {
  y <- rnorm(n_ex, mu_ex, sigma_ex)
  var(y)  # R 的 var() 默认用 n-1
})

cat("sigma^2 真实值:    ", sigma_ex^2, "\n")
sigma^2 真实值:     9 
cat("E[S_n^2] 模拟值:  ", round(mean(s2_samples), 3), "\n")
E[S_n^2] 模拟值:   9.007 
cat("确认: 无偏! | Bestatigt: Unbiased!\n")
确认: 无偏! | Bestatigt: Unbiased!

  1. 给出 KL 散度的定义，说明为什么 \(KL(g, f) \neq KL(f, g)\)（除非 \(f = g\)）。

  2. 给定样本 \(x = (2,2,3,2,3,0,2,1,4,3)^\top\)，假设 \(X_i \sim Bin(4, 0.5)\)。计算经验分布 \(g(\cdot)\) 与假设分布 \(f(\cdot)\) 之间的 KL 散度。

# Exercise 2b
x_sample <- c(2, 2, 3, 2, 3, 0, 2, 1, 4, 3)

tab <- table(factor(x_sample, levels = 0:4)) / length(x_sample)
g_emp <- as.numeric(tab)

f_bin <- dbinom(0:4, size = 4, prob = 0.5)

cat("k:     ", 0:4, "\n")
k:      0 1 2 3 4 
cat("g(k):  ", g_emp, "\n")
g(k):   0.1 0.1 0.4 0.3 0.1 
cat("f(k):  ", round(f_bin, 4), "\n\n")
f(k):   0.0625 0.25 0.375 0.25 0.0625 
# KL(g, f) — 注意 g(k)=0 时该项贡献为 0
kl_val_safe <- sum(ifelse(g_emp > 0, g_emp * log(g_emp / f_bin), 0))
cat("KL(经验分布, Bin(4,0.5)) =", round(kl_val_safe, 4), "\n")

  1. 在独立性假设下，给出一个关于总重期望值对称的区间，使得 95% 的情况下毛重落入该区间。

# Exercise 4a：总重的分布

Einleitung: Die Reise dieses Kapitels auf einen Blick

在当今世界，我们越来越频繁地观测到极端事件：极端干旱、极端降雨、河流的极端水位。前几章中我们所依赖的渐近推断工具（如中心极限定理）是为分布的”中心地带”设计的。然而当我们的兴趣转向分布的尾部时，这些经典工具将不再适用。幸运的是，极端事件拥有自己的极限定理，正如中心极限定理为均值提供了正态分布的理论依据，极值定理为最大值提供了极值分布的理论依据。

In der heutigen Welt beobachten wir immer mehr Extremereignisse. Die asymptotischen Werkzeuge wie der zentrale Grenzwertsatz gelten nur fur die Mitte der Verteilung. Fur die Rander gibt es jedoch eigene Grenzwertsatze – die Extremwerttheorie.

本章分为两个核心部分：

  1. 罕见事件 – 当事件发生概率极低（如 \(y = 0\)），渐近方法失效，精确推断如何补救
  2. 极端数据 – 分布尾部的极限定理，广义极值分布（GEV），参数估计与应用

Seltene Ereignisse

罕见事件是指发生频率很低和/或概率很小的事件。它们可以是极端事件（如导致洪水的高水位、极端热浪），也可以是极为罕见的事件（如核电站事故或飞机坠毁）。本节我们聚焦于事件发生次数极少的情形。

Tip黑天鹅的比喻 | Die Schwarze-Schwan-Metapher

想象一位鸟类学家来到一个湖边，观察了 10 只天鹅，全部是白色的。他能否因此断定”这个湖里没有黑天鹅”？当然不能。同样，在一个小样本中没有观测到某事件，并不意味着该事件的发生概率为零。罕见事件推断的核心挑战就在于：如何在”零观测”的情况下，对事件的真实概率给出有意义的估计和置信区间。

Stellen Sie sich einen Ornithologen vor, der 10 Schwane beobachtet – alle weiss. Kann er behaupten, es gebe keine schwarzen Schwane? Naturlich nicht. Ebenso bedeutet das Nichtbeobachten eines Ereignisses nicht, dass es unmoeglich ist.

1.1 二项分布的动机案例

Motivierendes Beispiel: Binomialverteilung

我们用一个简单的例子来引出核心思想。假设我们对学生中药物成瘾的比例 \(\pi\) 感兴趣。我们抽取了 \(n\) 名学生，每人回答是否成瘾（\(Y = 1\) 表示是，\(Y = 0\) 表示否）。为简单起见，假设学生都如实回答。现在假设样本量很小，\(n = 10\)，且没有一个学生回答”是”，即 \(y_i = 0\)（\(i = 1, \ldots, 10\)）。

在二项分布假设下，对数似然函数为：

\[ l(\pi) = 10 \log(1 - \pi) \]

这是 \(\pi\) 的递减函数，其最大值在 \(\hat{\pi} = 0\) 处取到。

1.2 渐近方法的失败

Versagen der asymptotischen Methoden

注意，此时对数似然函数在最大值处的一阶导数不等于零 – 最大值出现在参数空间的边界上，而不是内部。这与第 4 章推导的渐近结论相矛盾，后者要求 \(\partial l(\hat{\theta}) / \partial \theta = 0\) 作为推导的基础。

如果我们盲目地套用第 4 章的渐近结果，会得到毫无意义的推断：由于 \(\hat{\pi} = 0\)，估计的方差也是零，这反过来意味着 \(\pi = 0\) 是精确已知的。然而这显然不对 – 仅仅因为我们没有在样本中观察到成瘾学生，并不意味着不存在成瘾学生。

Tip放大镜与原子的比喻 | Die Lupe-und-Atom-Metapher

渐近推断方法就像一把普通放大镜 – 它在观察常见尺度的物体时效果很好，但如果你试图用它观察原子级别的微小事物，它就完全失效了。中心极限定理和似然比的渐近卡方分布是为”正常规模”的数据设计的，当事件极其罕见时，我们需要更精密的工具。

Die asymptotische Theorie ist wie eine Lupe – sie funktioniert gut fur normale Grossenordnungen, versagt aber bei atomaren Skalen. Fur seltene Ereignisse brauchen wir prazisere Werkzeuge.

作为一种尝试，我们可以利用似然比检验的渐近结果。根据第 4 章公式 (4.19)，有

\[ 2[l(\hat{\pi}) - l(\pi)] \sim \chi^2_1 \]

因此，对 \(H_0 : \pi = \pi_0\) 的检验在下式成立时拒绝原假设：

\[ l(\hat{\pi}) - l(\pi) > \frac{\chi^2_{1, 1-\alpha}}{2} \]

由此导出的近似置信区间为

\[ \left[0, \; \exp\left(-\frac{\chi^2_{1, 1-\alpha}}{2n}\right)\right] \]

但必须强调，这个区间没有数学上的严格证明，实际上它过于狭窄。换言之，第 4 章的渐近论证不成立，因为 \(\partial l(\hat{\theta}) / \partial \theta \neq 0\)。

1.3 Clopper-Pearson 精确置信区间

Exakte Konfidenzintervalle nach Clopper und Pearson

二项分布的结构允许我们计算精确置信区间，从而即使在罕见事件情形下也能进行有效的统计推断。这一方法由 Clopper 和 Pearson (1934) 提出。

其核心思想是：假设在 \(n\) 次试验中观测到 \(y = 0\)。我们可以计算，对于给定的 \(n\) 但不同的 \(\pi\) 值，观测到 \(Y = 0\) 的概率。然后，我们寻找使得这个概率等于 \(\alpha/2\) 的阈值 \(\pi_r\)。

一般构造原则如下。对于 \(Y = y\)，置信区间的下界为

\[ c_l = \begin{cases} 0 & \text{若 } y = 0 \\ \pi_l \text{ 满足 } P(Y \geq y;\, \pi_l) = \frac{\alpha}{2} & \text{其他} \end{cases} \]

置信区间的上界为

\[ c_r = \begin{cases} 1 & \text{若 } y = n \\ \pi_r \text{ 满足 } P(Y \leq y;\, \pi_r) = \frac{\alpha}{2} & \text{其他} \end{cases} \]

这个区间称为精确置信区间（exaktes Konfidenzintervall）。

Tip量身定制的西装比喻 | Die Massanzug-Metapher

渐近置信区间就像成衣 – 对大多数人（大样本、常见事件）穿着合身，但对极端身材（极小样本、罕见事件）就合不上了。Clopper-Pearson 精确置信区间则像量身定制的西装 – 它精确地利用了二项分布的形状，为每种具体情形提供恰到好处的覆盖。

Das asymptotische KI ist wie Konfektionskleidung – passt fur die meisten, aber nicht fur Extreme. Das Clopper-Pearson-KI ist ein Massanzug, der exakt auf die Binomialverteilung zugeschnitten ist.

在实际计算中，Clopper-Pearson 精确置信区间可以通过 Beta 分布的分位数函数方便地实现：下界为 \(\text{qbeta}(\alpha/2,\, y,\, n - y + 1)\)，上界为 \(\text{qbeta}(1 - \alpha/2,\, y + 1,\, n - y)\)。

# Clopper-Pearson 精确置信区间的 R 实现
clopper_pearson_ci <- function(y, n, alpha = 0.05) {
  lower <- ifelse(y == 0, 0, qbeta(alpha / 2, y, n - y + 1))
  upper <- ifelse(y == n, 1, qbeta(1 - alpha / 2, y + 1, n - y))
  return(c(lower = lower, upper = upper))
}

1.4 泊松分布的精确推断

Exakte Inferenz fur die Poisson-Verteilung

类似的思路也适用于泊松分布。假设我们有 \(n\) 个独立同分布的泊松随机变量

\[ Y_i \sim \text{Poisson}(\lambda), \quad \text{i.i.d.} \]

但我们观测到的是罕见事件，即 \(\lambda\) 极小。在最极端的情形下，我们根本没有观测到事件发生，即 \(y_i = 0\)（\(i = 1, \ldots, n\)）。数据清楚地表明 \(\lambda\) 很小，但并不意味着 \(\lambda = 0\)（尽管这是最大似然估计的值）。因此，我们需要不依赖渐近论证的置信区间。

首先考察似然函数。我们得到

\[ L(\lambda) = \prod_{i=1}^{n} \frac{\lambda^{y_i}}{y_i!} e^{-\lambda} \propto \lambda^{\sum_{i=1}^{n} y_i} e^{-n\lambda} \]

这表明 \(Y = \sum_{i=1}^{n} Y_i\) 服从参数为 \(n\lambda\) 的泊松分布，因此我们可以针对单个观测 \(Y \sim \text{Poisson}(\lambda_n)\)（其中 \(\lambda_n = n\lambda\)）推导置信区间。如果我们希望得到 \(\lambda\)（而非 \(\lambda_n\)）的置信区间，只需将区间端点除以 \(n\) 即可。

泊松分布的精确置信区间可以追溯到 Garwood (1936)。对于 \(\lambda_n\) 的置信区间，下界由下式给出

\[ c_l = \begin{cases} 0 & \text{若 } Y = 0 \\ \lambda_l \text{ 满足 } P(Y \geq y;\, \lambda_n = \lambda_l) = \frac{\alpha}{2} & \text{其他} \end{cases} \]

上界为

\[ c_r = \lambda_r \text{ 满足 } P(Y \leq y;\, \lambda_n = \lambda_r) = \frac{\alpha}{2} \]

在实际计算中，Garwood 精确置信区间可以通过卡方分布的分位数实现：下界为 \(\chi^2_{2y,\, \alpha/2} / 2\)（\(y > 0\) 时），上界为 \(\chi^2_{2(y+1),\, 1-\alpha/2} / 2\)。

garwood_ci <- function(y, alpha = 0.05) {
  lower <- ifelse(y == 0, 0, qchisq(alpha / 2, 2 * y) / 2)
  upper <- qchisq(1 - alpha / 2, 2 * (y + 1)) / 2
  return(c(lower = lower, upper = upper))
}

2.2 最大值的分布

Verteilung des Maximums

让我们形式化地定义上面的研究对象。假设经典设定

\[ Y_i \sim F(\cdot) \quad \text{i.i.d.}, \quad i = 1, \ldots, n \]

我们的兴趣在于

\[ Y_{\max, n} = \max\{Y_1, \ldots, Y_n\} \]

的分布。基于独立同分布假设，不难推导出最大值的分布函数：

\[ F_{Y_{\max, n}}(y) = P(Y_{\max, n} \leq y) = P(Y_1 \leq y, \ldots, Y_n \leq y) = P(Y_1 \leq y) \cdots P(Y_n \leq y) = F^n(y) \]

2.3 为什么直接取极限会退化

Warum der direkte Grenzwert degeneriert

如果我们直接取极限 \(n \to \infty\)，不会得到有用的极限分布。考虑两种情形：

情形一：支撑集有界。若存在一个最大值 \(c\)，使得 \(c = \sup\{y : F(y) < 1\}\)，则

\[ \lim_{n \to \infty} F_{Y_{\max, n}}(y) = \begin{cases} 0 & \text{若 } y < c \\ 1 & \text{若 } y \geq c \end{cases} \]

极限分布退化为集中在一点 \(c\) 的分布 – 对于无穷多数据，最大值就等于上界 \(c\)。

情形二：支撑集无界。此时随着 \(n\) 增大，最大值会越来越大，\(F_{Y_{\max, n}}(y)\) 变得退化。

2.4 标准化与极值定理

Standardisierung und der Extremwertsatz

Tip显微镜调焦的比喻 | Die Mikroskop-Metaphor

直接观察 \(F^n(y)\) 的极限就像把显微镜的放大倍率固定不动却想观察越来越小的物体 – 你什么也看不清。标准化序列 \(a_n\) 和 \(b_n\) 就是在不断调整显微镜的焦距和放大倍率：\(b_n\) 调整焦点位置（平移），\(a_n\) 调整放大倍率（缩放）。只有不断调整，才能在极限下看到清晰的”图像”（非退化分布）。

类比中心极限定理中的 \(\sqrt{n}\)：如果不除以 \(\sqrt{n}\)，样本均值的 \(n\) 倍会趋向无穷；除以 \(\sqrt{n}\) 后才得到标准正态极限。

Die Standardisierungssequenzen \(a_n\) und \(b_n\) sind wie die Fokussierung eines Mikroskops – ohne Anpassung wird das Bild unscharf. Erst die richtige Skalierung liefert ein nicht-degeneriertes Grenzbild.

我们假设存在序列 \(a_n\) 和 \(b_n\)（可以依赖于样本量），使得

\[ \frac{Y_{\max, n} - b_n}{a_n} \]

收敛到某个非退化分布。因此

\[ \lim_{n \to \infty} P\left(\frac{Y_{\max, n} - b_n}{a_n} \leq y\right) = \lim_{n \to \infty} P(Y_{\max, n} \leq a_n y + b_n) = \lim_{n \to \infty} F^n(a_n y + b_n) = H(y) \]

其中 \(y \in \mathbb{R}\)，\(H\) 是某个非退化分布。后者被称为极值分布，如下面的结果所示。

Important性质 10.1：极值定理 | Eigenschaft 10.1: Extremwertsatz

设 \(\{Y_i\}_{i=1}^{\infty}\) 为 i.i.d. 随机变量，\(Y_{\max, n} := \max(Y_1, \ldots, Y_n)\)。若存在实数序列 \(\{a_n\}_{n=1}^{\infty}\)，\(\{b_n\}_{n=1}^{\infty}\) 使得

\[ \frac{Y_{\max, n} - b_n}{a_n} \xrightarrow{d} Z \quad \text{当 } n \to \infty \]

则 \(Z\) 服从以下三种分布之一：

1. 极值分布或 Gumbel 分布：

\[ H_1(z) = \exp(-\exp(-z)), \quad z \in \mathbb{R} \]

2. （反转的）Weibull 分布：

\[ H_2(z) = \begin{cases} \exp(-(- z)^{1/k}) & \text{若 } z \leq 0 \\ 1 & \text{若 } z > 0 \end{cases} \]

3. Frechet-Pareto 分布：

\[ H_3(z) = \begin{cases} 0 & \text{若 } z < 0 \\ \exp(-z^{-1/k}) & \text{若 } z \geq 0 \end{cases} \]

我们可以将性质 10.1 理解为分布尾部的极限定理。换言之，如果关注的不是分布的中心（均值）而是尾部（极大值或极小值），也存在极限定理和自然的分布模型。

这一方法论可以追溯到 Gumbel (1958)，一位德国统计学家。他在政治上遭受迫害，在二战期间逃离纳粹德国前往美国。在那里，他研究了极值问题，如今被视为极值理论的奠基人。更多关于 Gumbel 生平和贡献的细节可参见 Rendtel et al. (2021)。

2.5 广义极值分布 (GEV)

Definition 10.1: Die verallgemeinerte Extremwertverteilung (GEV)

三种分布 \(H_1(\cdot)\)、\(H_2(\cdot)\) 和 \(H_3(\cdot)\) 可以统一写成一种形式，称为广义极值分布（Generalized Extreme Value distribution, GEV）。

Important定义 10.1：广义极值分布 | Definition 10.1: Verallgemeinerte Extremwertverteilung

GEV 有三个参数：位置参数 \(\mu\)、尺度参数 \(\sigma\) 和形状参数 \(\gamma\)。其分布函数为

\[ H(z) = \begin{cases} \exp\left(-(1 + \gamma z)^{-1/\gamma}\right) & \text{若 } \gamma \neq 0 \\ \exp(-\exp(-z)) & \text{若 } \gamma = 0 \end{cases} \]

其中 \(z = (x - \mu) / \sigma\)。

对 (10.2) 中的参数进行解读，与其他分布类似。参数 \(\mu\) 给出位置参数，\(\sigma\) 是尺度参数。形状参数 \(\gamma\) 决定了分布的类型：

  1. \(\gamma = 0\) 给出 Gumbel 分布 \(H_1(\cdot)\)
  2. \(\gamma > 0\) 给出（反转的）Weibull 分布 \(H_2(\cdot)\)
  3. \(\gamma < 0\) 给出 Frechet-Pareto 分布 \(H_3(\cdot)\)
Tip三种人格的比喻 | Die drei Personlichkeitstypen-Metapher

把三种极值分布想象成三种不同的”尾部人格”：

Gumbel（\(\gamma = 0\)） 是”温和型” – 尾部衰减速度适中（指数级），就像正态分布的尾部。大多数自然现象的极值属于这一类。

Weibull（\(\gamma > 0\)） 是”有限型” – 存在一个上界，极值永远不会超过某个最大值。想象一个有天花板的房间，球再怎么弹也不会超过天花板。

Frechet-Pareto（\(\gamma < 0\)） 是”重尾型” – 尾部衰减极慢（幂律衰减），极端事件比你预期的更常发生。想象一个没有天花板的房间，球偶尔能弹到惊人的高度。

Gumbel = mittlerer Schwanz, Weibull = beschrankter Schwanz, Frechet-Pareto = schwerer Schwanz.

GEV 的参数化开辟了使用最大似然法估计参数的可能性，进而估计极值分布的类型。这体现了极值分布方法的普适性和实用性。该领域已发展成熟，更多细节可参考 Coles et al. (2001) 或 Haan and Ferreira (2006)。

dgev_manual <- function(x, mu, sigma, gamma) {
  z <- (x - mu) / sigma
  if (abs(gamma) < 1e-8) {
    # Gumbel
    t_val <- exp(-z)
    dens <- (1 / sigma) * t_val * exp(-t_val)
  } else {
    t_val <- 1 + gamma * z
    valid <- t_val > 0
    dens <- rep(0, length(x))
    dens[valid] <- (1 / sigma) * t_val[valid]^(-1/gamma - 1) * 
                   exp(-t_val[valid]^(-1/gamma))
  }
  return(dens)
}

2.6 GEV 的最大似然估计

Maximum-Likelihood-Schatzung der GEV

GEV 分布的对数似然函数为：

对于 \(\gamma \neq 0\)：

\[ l(\mu, \sigma, \gamma) = -n \log \sigma - \left(\frac{1}{\gamma} + 1\right) \sum_{i=1}^{n} \log\left(1 + \gamma \frac{y_i - \mu}{\sigma}\right) - \sum_{i=1}^{n} \left(1 + \gamma \frac{y_i - \mu}{\sigma}\right)^{-1/\gamma} \]

条件是 \(1 + \gamma (y_i - \mu) / \sigma > 0\) 对所有 \(i\) 成立。

对于 \(\gamma = 0\)（Gumbel 情形）：

\[ l(\mu, \sigma) = -n \log \sigma - \sum_{i=1}^{n} \frac{y_i - \mu}{\sigma} - \sum_{i=1}^{n} \exp\left(-\frac{y_i - \mu}{\sigma}\right) \]

这些对数似然函数可以通过数值优化方法（如 R 中的 optim()）最大化，从而获得参数估计。

# GEV 负对数似然函数
nll_gev <- function(par, data) {
  mu <- par[1]
  sigma <- par[2]
  gamma <- par[3]
  n <- length(data)
  
  if (sigma <= 0) return(1e10)
  
  z <- (data - mu) / sigma
  
  if (abs(gamma) < 1e-8) {
    # Gumbel 情形
    nll <- n * log(sigma) + sum(z) + sum(exp(-z))
  } else {
    t_val <- 1 + gamma * z
    if (any(t_val <= 0)) return(1e10)
    nll <- n * log(sigma) + (1/gamma + 1) * sum(log(t_val)) + sum(t_val^(-1/gamma))
  }
  return(nll)
}

# GEV 拟合函数
fit_gev <- function(data, start = NULL) {
  if (is.null(start)) {
    # Gumbel 矩估计作为初始值
    mu0 <- mean(data) - 0.5772 * sd(data) * sqrt(6) / pi
    sigma0 <- sd(data) * sqrt(6) / pi
    gamma0 <- 0.01
    start <- c(mu0, sigma0, gamma0)
  }
  
  result <- optim(start, nll_gev, data = data, method = "Nelder-Mead",
                  hessian = TRUE,
                  control = list(maxit = 10000))
  
  se <- tryCatch(sqrt(diag(solve(result$hessian))), error = function(e) rep(NA, 3))
  
  list(
    par = result$par,
    se = se,
    nll = result$value,
    convergence = result$convergence
  )
}

# GEV 分布函数
pgev_manual <- function(x, mu, sigma, gamma) {
  z <- (x - mu) / sigma
  if (abs(gamma) < 1e-8) {
    p <- exp(-exp(-z))
  } else {
    t_val <- 1 + gamma * z
    valid <- t_val > 0
    p <- rep(0, length(x))
    p[valid] <- exp(-t_val[valid]^(-1/gamma))
    if (gamma > 0) {
      p[!valid & z < -1/gamma] <- 0
    } else {
      p[!valid & z > -1/gamma] <- 1
    }
  }
  return(p)
}

2.8 证明概要

Beweisskizze fur Eigenschaft 10.1

为了给出性质 10.1 的证明概要，将 \(Y_{\max, n}\) 视为矩阵形式数据的最大值是有帮助的。将 \(Y\) 用双下标写成矩阵形式：

\[ \begin{pmatrix} Y_{11} & Y_{12} & \cdots & Y_{1K} \\ Y_{21} & Y_{22} & \cdots & Y_{2K} \\ \vdots & \vdots & \ddots & \vdots \\ Y_{T1} & Y_{T2} & \cdots & Y_{TK} \end{pmatrix} \]

其中 \(n = T \times K\)。例如，\(Y_{t1}, \ldots, Y_{tK}\) 可以视为第 \(t\) 年的每日温度。定义

\[ Y_{t, \max, K} = \max\{Y_{t1}, \ldots, Y_{tK}\} \]

为行最大值（在例子中即年度最高温度）。总最大值为

\[ Y_{\max, n} = \max_{1 \leq t \leq T}\left\{\max_{1 \leq k \leq K}(Y_{tk})\right\} = \max_{1 \leq t \leq T}\{Y_{t, \max, K}\} \]

这也被称为取”分块最大值”（Blockmaxima），其中分块在这里对应年份。

如果存在标准化序列 \((a_n, b_n)\) 使得 \((Y_{\max, n} - b_n)/a_n\) 收敛到分布 \(Z\)，那么相同的极限也应对行最大值成立。假设独立性，所有行最大值具有相同的分布 \(F_{Y_{\max, K}}\)。利用上式我们得到

\[ F_{Y_{\max, TK}}(y) = F_{Y_{\max, K}}^T(y) \]

两个序列经过适当标准化后收敛到相同的分布函数。应用标准化，可以推导出变量 \(Z\) 的分布函数具有以下性质：

\[ F_Z(b_T + a_T z) = [F_Z(z)]^T \]

可以验证上述极值分布满足这一性质。以 Gumbel 分布为例，设 \(a_T = 1\) 和 \(b_T = \log(T)\)，则

\[ H_1(y - \log(T)) = \exp(-\exp(-(y - \log(T)))) = \exp(-\exp(-y) \cdot T) = [\exp(-\exp(-y))]^T = H_1(y)^T \]

类似地，可以对其他极值分布验证此性质，从而完成证明。

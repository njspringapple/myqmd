1 可微性的”赠品”：连续性与线性性

Geschenke der Differenzierbarkeit: Stetigkeit und Linearität

Note🎯 本节你将学到 | Was Sie in diesem Abschnitt lernen
  • 可微 \(\implies\) 连续——但反过来不成立
  • 导数算子具有 线性性：可以拆分复杂梯度
  • Differenzierbarkeit \(\implies\) Stetigkeit, aber nicht umgekehrt
  • Der Ableitungsoperator ist linear: komplexe Gradienten lassen sich zerlegen

1.1 可微 ⟹ 连续

Differenzierbar ⟹ stetig

定理 | Satz： 设 \(\mathbb{D}\) 和 \(\mathbb{E}\) 为赋范空间，\(f:\mathbb{D}\to\mathbb{E}\) 在 \(\mathbf{x}\in\mathbb{D}\) 处可微，则 \(f\) 在 \(\mathbf{x}\) 处连续。

证明 | Beweis：

由可微性定义，\(D_\mathbf{x}f\) 是有界线性算子，因此存在常数 \(M\) 使得

\[ \|D_\mathbf{x}f(\mathbf{h})\| \le M\|\mathbf{h}\| = O(\|\mathbf{h}\|) \]

于是

\[ f(\mathbf{x}+\mathbf{h}) = f(\mathbf{x}) + \underbrace{D_\mathbf{x}f(\mathbf{h})}_{O(\|\mathbf{h}\|)} + o(\|\mathbf{h}\|) = f(\mathbf{x}) + O(\|\mathbf{h}\|) \]

当 \(\|\mathbf{h}\|\to 0\) 时，\(f(\mathbf{x}+\mathbf{h})\to f(\mathbf{x})\)。 \(\square\)

Important💡 关键洞察 | Schlüsselerkenntnis

可微性 \(\implies\) 连续性。这意味着：如果损失函数可微，它就不会”跳跃”——梯度下降沿着一条 连续的 曲面走，不会突然掉进悬崖。

Wenn die Verlustfunktion differenzierbar ist, „springt” sie nicht — der Gradientenabstieg bewegt sich auf einer stetigen Oberfläche.

1.2 反例：连续但不可微

Gegenbeispiel: stetig, aber nicht differenzierbar

经典反例：\(f(x)=|x|\)。

  • 在 \(x=0\) 处 连续：\(f(0+h)=|h|\to 0=f(0)\)。✓

  • 在 \(x=0\) 处 不可微：左右极限不同。✗

\[ \lim_{h\to 0^+}\frac{|h|}{h}=1, \qquad \lim_{h\to 0^-}\frac{|h|}{h}=-1 \]

1.3 导数算子的线性性

Linearität des Ableitungsoperators

定理 | Satz： 设 \(f,g:\mathbb{D}\to\mathbb{E}\) 在 \(\mathbf{x}\) 处可微，则对所有 \(c_1,c_2\in\mathbb{R}\)：

\[ D_\mathbf{x}(c_1 f + c_2 g) = c_1 \, D_\mathbf{x}f + c_2 \, D_\mathbf{x}g \]

这意味着：求导可以”穿过”加法和常数乘法。把它套到不同的空间上，就得到我们熟悉的各种形式：

空间 / Räume 形式 / Form
\(f,g:\mathbb{R}\to\mathbb{R}\) \((c_1 f+c_2 g)'=c_1 f'+c_2 g'\)
\(f,g:\mathbb{R}^n\to\mathbb{R}\) \(\nabla(c_1 f+c_2 g)=c_1\nabla f+c_2\nabla g\)
\(f,g:\mathbb{R}^n\to\mathbb{R}^m\) \(\mathbf{J}_{c_1 f+c_2 g}=c_1\mathbf{J}_f+c_2\mathbf{J}_g\)
\(f,g:\mathbb{R}^{m\times n}\to\mathbb{R}\) \(\frac{\partial(c_1 f+c_2 g)}{\partial\mathbf{X}}=c_1\frac{\partial f}{\partial\mathbf{X}}+c_2\frac{\partial g}{\partial\mathbf{X}}\)

1.4 示例：线性性的威力

Beispiel: Die Kraft der Linearität

已知 \(f(\mathbf{x})=\mathbf{x}^\top\mathbf{x}\) 和 \(g(\mathbf{x})=\mathbf{a}^\top\mathbf{x}\)（其中 \(\mathbf{a}=(1,2)^\top\)）。在 \(\mathbf{x}=(1,-1)^\top\) 处计算 \(h(\mathbf{x})=3f(\mathbf{x})-2g(\mathbf{x})\) 的梯度。

解 | Lösung：

分别求梯度：\(\nabla f(\mathbf{x})=2\mathbf{x}^\top\)，\(\nabla g(\mathbf{x})=\mathbf{a}^\top=(1,\;2)\)。

由线性性：

\[ \nabla h(\mathbf{x}) = 3\nabla f(\mathbf{x})-2\nabla g(\mathbf{x}) = 3(2x_1,\;2x_2)-2(1,\;2) = (6x_1-2,\;6x_2-4) \]

代入 \(\mathbf{x}=(1,-1)^\top\)：

\[ \nabla h\!\left(\begin{pmatrix}1\\-1\end{pmatrix}\right) = (6\cdot 1-2,\;6\cdot(-1)-4) = (4,\;-10) \]

2 乘积法则：处理复合表达式

Produktregel: Zusammengesetzte Ausdrücke handhaben

Note🎯 本节你将学到 | Was Sie lernen
  • 一般化的 乘积法则：适用于任意双线性映射
  • 从一维到矩阵的各种特化形式
  • 经典应用：二次型 \(\mathbf{x}^\top\!\mathbf{A}\mathbf{x}\) 的求导

2.1 一般乘积法则

Allgemeine Produktregel

定理 | Satz： 设 \(\langle\cdot,\cdot\rangle:\mathbb{E}\times\mathbb{E}\to\mathbb{B}\) 为双线性映射（Bilinearform），\(b(\mathbf{x})=\langle f(\mathbf{x}),\,g(\mathbf{x})\rangle\)。若 \(f,g\) 在 \(\mathbf{x}\) 处可微，则

\[ D_\mathbf{x}b(\mathbf{h}) = \langle D_\mathbf{x}f(\mathbf{h}),\;g(\mathbf{x})\rangle + \langle f(\mathbf{x}),\;D_\mathbf{x}g(\mathbf{h})\rangle \]

Tip🔍 证明 | Beweis（点击展开）

将 \(b(\mathbf{x}+\mathbf{h})\) 展开：

\[ b(\mathbf{x}+\mathbf{h})=\langle f(\mathbf{x}+\mathbf{h}),\;g(\mathbf{x}+\mathbf{h})\rangle \]

由 \(f,g\) 可微，分别代入 \(f(\mathbf{x}+\mathbf{h})=f(\mathbf{x})+D_\mathbf{x}f(\mathbf{h})+o(\|\mathbf{h}\|)\) 和 \(g\) 的类似展开，然后利用双线性性展开得到 9 项。其中：

  • \(\langle f(\mathbf{x}),g(\mathbf{x})\rangle = b(\mathbf{x})\)（零阶项）
  • \(\langle D_\mathbf{x}f(\mathbf{h}),g(\mathbf{x})\rangle + \langle f(\mathbf{x}),D_\mathbf{x}g(\mathbf{h})\rangle\)（一阶项 = \(D_\mathbf{x}b(\mathbf{h})\)）
  • 剩余项：\(\langle D_\mathbf{x}f(\mathbf{h}),D_\mathbf{x}g(\mathbf{h})\rangle = O(\|\mathbf{h}\|^2) = o(\|\mathbf{h}\|)\) 以及包含 \(o(\|\mathbf{h}\|)\) 的项，都被吸收进 \(o(\|\mathbf{h}\|)\)。\(\square\)

2.2 特化形式一览

Spezialfälle im Überblick

乘积法则在不同空间上的”长相”：

空间 / Räume 乘积 / Produkt 法则 / Regel
\(f,g:\mathbb{R}\to\mathbb{R}\) \(fg\) \(b'=f'g+fg'\)
\(f,g:\mathbb{R}^n\to\mathbb{R}\) \(fg\) \(\nabla b=g\nabla f+f\nabla g\)
\(f,g:\mathbb{R}^n\to\mathbb{R}^m\) \(f^\top g\) \(\nabla b=g^\top\mathbf{J}_f+f^\top\mathbf{J}_g\)
\(\mathbf{F},\mathbf{G}:\mathbb{R}\to\mathbb{R}^{m\times n}\) \(\mathrm{tr}[\mathbf{F}\mathbf{G}]\) \(b'=\mathrm{tr}\!\big[\mathbf{F}'\mathbf{G}\big]+\mathrm{tr}\!\big[\mathbf{F}\mathbf{G}'\big]\)

2.3 经典应用：二次型的导数

Klassische Anwendung: Ableitung der quadratischen Form

设 \(b(\mathbf{x})=\mathbf{x}^\top\!\mathbf{A}\mathbf{x}\)，其中 \(f(\mathbf{x})=\mathbf{x}\)，\(g(\mathbf{x})=\mathbf{A}\mathbf{x}\)。

计算所需的 Jacobi 矩阵：\(\mathbf{J}_f(\mathbf{x})=\mathbf{I}_n\)，\(\mathbf{J}_g(\mathbf{x})=\mathbf{A}\)。

由乘积法则（第三行）：

\[ \nabla b(\mathbf{x}) = g(\mathbf{x})^\top\mathbf{J}_f(\mathbf{x}) + f(\mathbf{x})^\top\mathbf{J}_g(\mathbf{x}) = (\mathbf{A}\mathbf{x})^\top\mathbf{I}_n + \mathbf{x}^\top\mathbf{A} = \mathbf{x}^\top(\mathbf{A}^\top+\mathbf{A}) \]

若 \(\mathbf{A}\) 对称（\(\mathbf{A}=\mathbf{A}^\top\)），则

\[ \boxed{\nabla(\mathbf{x}^\top\!\mathbf{A}\mathbf{x}) = 2\mathbf{x}^\top\mathbf{A}} \]

3 链式法则：反向传播的数学基石

Kettenregel: Das mathematische Fundament der Backpropagation

Note🎯 本节你将学到 | Was Sie lernen
  • 一般化的 链式法则：函数复合时的求导法则
  • 在各种空间中的特化形式（标量→向量→矩阵）
  • 应用：逻辑回归的梯度推导

3.1 一般链式法则

Allgemeine Kettenregel

定理 | Satz： 设 \(f:\mathbb{D}_1\to\mathbb{D}_2\)，\(g:\mathbb{D}_2\to\mathbb{D}_3\)，令 \(\mathbf{y}=f(\mathbf{x})\)。若 \(f\) 在 \(\mathbf{x}\) 处可微，\(g\) 在 \(\mathbf{y}\) 处可微，则

\[ \boxed{D_\mathbf{x}(g\circ f) = D_\mathbf{y}g \circ D_\mathbf{x}f} \]

用一句话说 | In einem Satz： 复合函数的导数 = 外层导数 \(\circ\) 内层导数。

Tip🔍 证明 | Beweis（点击展开）

由 \(f\) 可微：

\[ f(\mathbf{x}+\mathbf{h}) = f(\mathbf{x}) + D_\mathbf{x}f(\mathbf{h}) + o(\|\mathbf{h}\|) \]

代入 \(g\)：

\[ (g\circ f)(\mathbf{x}+\mathbf{h}) = g\!\Big(f(\mathbf{x}) + \underbrace{D_\mathbf{x}f(\mathbf{h}) + o(\|\mathbf{h}\|)}_{\text{记为}\;\boldsymbol{\delta}}\Big) \]

由 \(g\) 在 \(\mathbf{y}=f(\mathbf{x})\) 处可微：

\[ = g(\mathbf{y}) + D_\mathbf{y}g(\boldsymbol{\delta}) + o(\|\boldsymbol{\delta}\|) \]

由 \(D_\mathbf{y}g\) 的线性性：

\[ D_\mathbf{y}g(\boldsymbol{\delta}) = D_\mathbf{y}g\!\big(D_\mathbf{x}f(\mathbf{h})\big) + D_\mathbf{y}g\!\big(o(\|\mathbf{h}\|)\big) \]

由 \(D_\mathbf{y}g\) 有界，\(D_\mathbf{y}g(o(\|\mathbf{h}\|))=o(\|\mathbf{h}\|)\)；又 \(\|\boldsymbol{\delta}\|=O(\|\mathbf{h}\|)\)，故 \(o(\|\boldsymbol{\delta}\|)=o(\|\mathbf{h}\|)\)。合并后：

\[ (g\circ f)(\mathbf{x}+\mathbf{h}) = (g\circ f)(\mathbf{x}) + (D_\mathbf{y}g\circ D_\mathbf{x}f)(\mathbf{h}) + o(\|\mathbf{h}\|) \quad \square \]

3.3 练习：欧几里得范数的梯度

Übung: Gradient der euklidischen Norm

设 \(h(\mathbf{x})=\|\mathbf{x}\|_2=\sqrt{\mathbf{x}^\top\mathbf{x}}\)。用链式法则求 \(\nabla h(\mathbf{x})\)。

解 | Lösung： 令 \(f(\mathbf{x})=\mathbf{x}^\top\mathbf{x}\)，\(g(t)=\sqrt{t}=t^{1/2}\)，则 \(h=g\circ f\)。

  • \(\nabla f(\mathbf{x})=2\mathbf{x}^\top\)（二次型求导）
  • \(g'(t)=\frac{1}{2}t^{-1/2}\)

由链式法则（\(f:\mathbb{R}^n\to\mathbb{R}\)，\(g:\mathbb{R}\to\mathbb{R}\)）：

\[ \nabla h(\mathbf{x}) = g'(f(\mathbf{x}))\,\nabla f(\mathbf{x}) = \frac{1}{2(\mathbf{x}^\top\mathbf{x})^{1/2}}\cdot 2\mathbf{x}^\top = \frac{\mathbf{x}^\top}{\|\mathbf{x}\|} \]

Tip🤔 想一想 | Denken Sie nach

\(\nabla h(\mathbf{x})=\frac{\mathbf{x}^\top}{\|\mathbf{x}\|}\) 就是 单位向量——范数增长最快的方向就是 \(\mathbf{x}\) 自身的方向。几何上完全合理！

Der Gradient der Norm ist der Einheitsvektor — die Richtung des steilsten Anstiegs der Norm ist die Richtung von \(\mathbf{x}\) selbst.

4.1 高阶 Fréchet 导数

Höhere Fréchet-Ableitungen

一阶导数 \(D_\mathbf{x}f\) 是线性映射。如果它本身关于 \(\mathbf{x}\) 也是可微的，我们就得到二阶导数——一个 双线性映射：

\[ D^2_\mathbf{x}f(\mathbf{h}_1,\mathbf{h}_2) = \lim_{\|\mathbf{h}\|\to 0}\frac{D_{\mathbf{x}+\mathbf{h}}f(\mathbf{h}_1)-D_\mathbf{x}f(\mathbf{h}_1)-D^2_\mathbf{x}f(\mathbf{h}_1,\mathbf{h})}{\|\mathbf{h}\|}=0 \]

一般地，\(k\) 阶导数 \(D^k_\mathbf{x}f\) 是一个 \(k\)-线性映射。随着阶数增长，涉及的对象变成 张量（tensor）。

4.2 Hesse 矩阵：二阶导数的矩阵表示

Hesse-Matrix: Matrixdarstellung der zweiten Ableitung

对于 \(f:\mathbb{R}^n\to\mathbb{R}\)，二阶导数可以用一个 \(n\times n\) 矩阵紧凑表示：

\[ \mathbf{H}_f(\mathbf{x})_{i,j} := \frac{\partial^2 f(\mathbf{x})}{\partial x_i\,\partial x_j} \in \mathbb{R}^{n\times n} \]

此时各阶导数的作用方式为：

  • 一阶：\(D_\mathbf{x}f(\mathbf{h})=\nabla f(\mathbf{x})\,\mathbf{h}\)
  • 二阶：\(D^2_\mathbf{x}f(\mathbf{h}_1,\mathbf{h}_2)=\mathbf{h}_1^\top\mathbf{H}_f(\mathbf{x})\,\mathbf{h}_2\)

Schwarz 定理 | Satz von Schwarz： 若 \(f\in\mathscr{C}^2\)（二阶连续可微），则

\[ \frac{\partial^2 f}{\partial x_i\,\partial x_j}=\frac{\partial^2 f}{\partial x_j\,\partial x_i} \qquad\Longrightarrow\qquad \mathbf{H}_f(\mathbf{x})=\mathbf{H}_f(\mathbf{x})^\top \]

Hesse 矩阵是 对称 的。这将需要计算的独立二阶偏导数从 \(n^2\) 减少到 \(\frac{n(n+1)}{2}\)。

4.3 几何意义：曲率与临界点分类

Geometrische Bedeutung: Krümmung und Klassifikation kritischer Punkte

Hesse 矩阵刻画了函数在某点附近的 局部曲率。在临界点 \(\mathbf{x}^*\)（即 \(\nabla f(\mathbf{x}^*)=\mathbf{0}\)）处：

\(\mathbf{H}_f(\mathbf{x}^*)\) 的性质 / Eigenschaft 临界点类型 / Typ 直觉 / Intuition
正定（所有特征值 \(>0\)） 局部极小值 所有方向都向上弯曲——碗底
负定（所有特征值 \(<0\)） 局部极大值 所有方向都向下弯曲——山顶
不定（有正有负特征值） 鞍点 有的方向上弯、有的下弯——马鞍

4.4 应用：Fisher 信息矩阵

Anwendung: Fisher-Informationsmatrix

\[ \mathbf{I}(\boldsymbol{\theta}) := \mathbb{E}\!\left[-\mathbf{H}_\ell(\boldsymbol{\theta})\right] = -\mathbb{E}\!\left[\frac{\partial^2\ell(\boldsymbol{\theta})}{\partial\theta_i\,\partial\theta_j}\right]_{i,j} \in \mathbb{R}^{p\times p} \]

Cramér-Rao 下界 | Cramér-Rao-Schranke：

\[ \mathrm{Var}(\hat{\boldsymbol{\theta}}) \xrightarrow{n\to\infty} \mathbf{I}(\boldsymbol{\theta})^{-1} \]

信息越多 = 方差越小 = 估计越精确。

5.2 一般 Taylor 展开

Allgemeine Taylor-Entwicklung

定理 | Satz： 设 \(f:V\to W\) 为 \(k\) 次连续 Fréchet 可微函数（\(f\in\mathscr{C}^k\)），则

\[ f(\mathbf{x}+\mathbf{h}) = f(\mathbf{x}) + \sum_{j=1}^{k}\frac{1}{j!}\,D^j_\mathbf{x}f(\underbrace{\mathbf{h},\ldots,\mathbf{h}}_{j\text{ 次}}) + o(\|\mathbf{h}\|^k) \]

Tip🔍 证明思路 | Beweisskizze（点击展开）
  1. 令 \(s:=\|\mathbf{h}\|\)，\(\mathbf{u}:=\mathbf{h}/s\)（单位向量）。
  2. 定义一维函数 \(\psi(w):=f(\mathbf{x}+w\mathbf{u})\)。
  3. 反复使用链式法则得 \(\psi^{(j)}(w)=D^j_{\mathbf{x}+w\mathbf{u}}f(\mathbf{u},\ldots,\mathbf{u})\)。
  4. 对 \(\psi\) 在 \(w=0\) 处作一维 Taylor 展开：\(\psi(s)=\sum_{j=0}^{k}\frac{s^j}{j!}\psi^{(j)}(0)+o(s^k)\)。
  5. 利用 \(D^j_\mathbf{x}f\) 的多线性性：\(D^j_\mathbf{x}f(\mathbf{u},\ldots,\mathbf{u})=\frac{1}{s^j}D^j_\mathbf{x}f(\mathbf{h},\ldots,\mathbf{h})\)。
  6. 代入 \(s=\|\mathbf{h}\|\) 即得结论。\(\square\)

5.3 \(f:\mathbb{R}^n\to\mathbb{R}\) 的展开公式

Entwicklungsformel für \(f:\mathbb{R}^n\to\mathbb{R}\)

这是机器学习中最常用的特化形式。将一般公式中的高阶导数用梯度和 Hesse 矩阵表示：

一阶近似（线性化 | Linearisierung）：

\[ f(\mathbf{x}+\mathbf{h}) \approx f(\mathbf{x}) + \nabla f(\mathbf{x})\,\mathbf{h} \]

二阶近似（二次近似 | Quadratische Approximation）：

\[ f(\mathbf{x}+\mathbf{h}) \approx f(\mathbf{x}) + \nabla f(\mathbf{x})\,\mathbf{h} + \frac{1}{2}\mathbf{h}^\top\mathbf{H}_f(\mathbf{x})\,\mathbf{h} \]

三阶近似：

\[ f(\mathbf{x}+\mathbf{h}) \approx f(\mathbf{x}) + \nabla f(\mathbf{x})\,\mathbf{h} + \frac{1}{2}\mathbf{h}^\top\mathbf{H}_f(\mathbf{x})\,\mathbf{h} + \frac{1}{6}\sum_{i,j,k}\frac{\partial^3 f}{\partial x_i\,\partial x_j\,\partial x_k}h_i h_j h_k \]

以展开点 \(\mathbf{x}_0\) 表示（令 \(\mathbf{h}=\mathbf{x}-\mathbf{x}_0\)）：

\[ f(\mathbf{x}) \approx f(\mathbf{x}_0) + \nabla f(\mathbf{x}_0)(\mathbf{x}-\mathbf{x}_0) + \frac{1}{2}(\mathbf{x}-\mathbf{x}_0)^\top\mathbf{H}_f(\mathbf{x}_0)(\mathbf{x}-\mathbf{x}_0) \]

6.1 从 Taylor 展开到 Newton 步

Von der Taylor-Entwicklung zum Newton-Schritt

问题 | Problem： 对 \(f\in\mathscr{C}^2\)，求 \(\mathbf{x}^*=\arg\min_\mathbf{x} f(\mathbf{x})\)。

思路 | Idee： 在当前点 \(\mathbf{x}^{(k)}\) 处用二阶 Taylor 多项式近似 \(f\)：

\[ f(\mathbf{x}) \approx T_2(\mathbf{x}) = f(\mathbf{x}^{(k)}) + \nabla f(\mathbf{x}^{(k)})(\mathbf{x}-\mathbf{x}^{(k)}) + \frac{1}{2}(\mathbf{x}-\mathbf{x}^{(k)})^\top\mathbf{H}_f(\mathbf{x}^{(k)})(\mathbf{x}-\mathbf{x}^{(k)}) \]

对这个 二次函数 求最小值——令梯度为零：

\[ \nabla T_2(\mathbf{x}) = \nabla f(\mathbf{x}^{(k)}) + (\mathbf{x}-\mathbf{x}^{(k)})^\top\mathbf{H}_f(\mathbf{x}^{(k)}) \overset{!}{=} \mathbf{0}^\top \]

解出：

\[ (\mathbf{x}-\mathbf{x}^{(k)})^\top = -\nabla f(\mathbf{x}^{(k)})\,\mathbf{H}_f(\mathbf{x}^{(k)})^{-1} \]

由此得到 Newton 迭代公式：

\[ \boxed{\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - \Big(\nabla f(\mathbf{x}^{(k)})\,\mathbf{H}_f(\mathbf{x}^{(k)})^{-1}\Big)^\top} \]

Tip🤔 对比梯度下降 | Vergleich mit Gradientenabstieg
方面 / Aspekt 梯度下降 Newton 法
用到的信息 一阶（梯度） 一阶 + 二阶（梯度 + Hesse）
每步计算量 \(O(Np)\) \(O(Np^2 + p^3)\)
收敛速度 线性收敛 二次收敛（接近最优时极快）
超参数 学习率 \(\eta\) 无（自适应步长）
适用规模 任意 \(p\) 不能太大

# 这是一个有两个局部极小值和一个鞍点的函数
# 为了演示 Newton 法的二次收敛，用一个强凸的二次+四次混合函数

f_obj <- function(x) 5*x[1]^2 + 10*x[2]^2 + 0.5*x[1]^4 + 0.5*x[2]^4 + 3*x[1]*x[2]

grad_obj <- function(x) {
  c(10*x[1] + 2*x[1]^3 + 3*x[2],
    20*x[2] + 2*x[2]^3 + 3*x[1])
}

hesse_obj <- function(x) {
  matrix(c(10 + 6*x[1]^2, 3,
           3, 20 + 6*x[2]^2), 2, 2)
}

# Newton 法
newton_run <- function(x_init, max_iter = 20, tol = 1e-12) {
  x <- x_init
  history <- matrix(NA, max_iter + 1, 2)
  loss_h <- numeric(max_iter + 1)
  history[1,] <- x
  loss_h[1] <- f_obj(x)
  
  for (i in 1:max_iter) {
    g <- grad_obj(x)
    H <- hesse_obj(x)
    delta <- solve(H, g)  # H^{-1} * g
    x <- x - delta
    history[i+1,] <- x
    loss_h[i+1] <- f_obj(x)
    if (sqrt(sum(g^2)) < tol) {
      history <- history[1:(i+1),]
      loss_h <- loss_h[1:(i+1)]
      break
    }
  }
  list(x = x, history = history, loss = loss_h)
}

# 梯度下降
gd_simple <- function(x_init, lr, max_iter = 200) {
  x <- x_init
  loss_h <- numeric(max_iter + 1)
  loss_h[1] <- f_obj(x)
  
  for (i in 1:max_iter) {
    g <- grad_obj(x)
    x <- x - lr * g
    loss_h[i+1] <- f_obj(x)
  }
  list(x = x, loss = loss_h)
}

Important💡 关键洞察 | Schlüsselerkenntnis

Newton 法利用了 曲率信息（Hesse 矩阵），所以它能自适应地调整步长和方向——在”碗壁陡”的地方走小步，在”碗壁缓”的地方走大步。代价是每步需要计算和求逆 Hesse 矩阵（\(O(p^3)\)），所以只适用于参数不太多的情形。

在机器学习/统计中的应用 | Anwendungen：

  • 最大似然估计：经典统计方法
  • 广义线性模型 (GLM)：IRLS/IWLS 算法本质上就是 Newton 法
  • 信赖域方法 (Trust Region)：现代优化器中 Newton 法的稳定化版本

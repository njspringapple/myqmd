在前面的章节中，我们讨论的模型大多涉及单个响应变量 \(Y\)，或者 \(Y\) 依赖于某些输入变量 \(x\) 的条件分布。现在，我们将目光转向一个全新的场景：观测到的变量 \(Y = (Y_1, \ldots, Y_p)\) 具有多个分量，这些分量之间彼此相互影响。与回归分析不同，这里不存在因果次序 – 我们不预设哪个变量是”因”、哪个是”果”，而是认为所有变量处于平等地位（auf gleicher Augenhöhe）。我们的目标是量化这些变量之间的关联（Assoziationen），特别是超越简单的两两相关性，探索更高维的依赖结构。

In diesem Kapitel betrachten wir multivariate Daten \(Y = (Y_1, \ldots, Y_p)\), deren Komponenten sich gegenseitig beeinflussen. Es gibt keine kausale Ordnung – alle Variablen stehen auf gleicher Augenhöhe. Unser Ziel ist die Quantifizierung von Assoziationen.

具体来说，我们将从离散数据出发，逐步构建完整的多元分析框架：

  1. 列联表的对数线性模型 – 离散多元数据、单元格计数、交互项
  2. 图模型 – 条件独立性、无向图、有向无环图 (DAG)
  3. 多元正态分布 – 浓度矩阵、条件独立的矩阵判据

1.1 从一维到多维：列联表的世界

Von einer Dimension zu vielen: Die Welt der Kontingenztafeln

Tip多维格子间的比喻 | Die Metapher des mehrdimensionalen Regalschranks

想象一个巨大的仓库货架系统。一维情形就是一排货架，每个格子对应一个类别。二维情形就像一面墙上的方格架，行代表一个变量、列代表另一个变量，每个格子存放满足两个条件的物品数量。三维情形就像一个立体的储物柜，有行、列、层三个维度。推广到 \(p\) 维，就是一个 \(p\) 维的”超级储物柜” – 虽然人类无法直接可视化，但数学上完全可以精确描述。每个格子里存放的物品数量就是单元格计数（Zellhäufigkeit）。

Stellen Sie sich ein mehrdimensionales Regalschrank vor: In jeder Zelle zählen wir, wie viele Beobachtungen gleichzeitig alle entsprechenden Kategorien erfüllen.

设 \(Y = (Y_1, \ldots, Y_p)\) 为 \(p\) 维随机向量，每个分量取离散值 \(Y_j \in \{1, \ldots, K_j\}\)，\(j = 1, \ldots, p\)。则 \(Y\) 的取值空间可以看作一个 \(p\) 维的长方体，其各边长为 \(K_1, K_2, \ldots, K_p\)。

1.2 符号约定

Notationskonvention

对于多元多观测数据，我们需要同时标注分量索引和观测索引。为避免双下标的混乱，本章采用以下记法：用括号上标表示观测编号。即 \(Y^{(i)} = (Y_1^{(i)}, \ldots, Y_p^{(i)})\) 表示第 \(i\) 个观测的 \(p\) 维随机向量，\(i = 1, \ldots, n\)。数据记为 \(Y^{(1)}, \ldots, Y^{(n)}\) 个独立多元观测。

单元格计数定义为：

\[ n_{k_1 k_2 \cdots k_p} = \sum_{i=1}^{n} \mathbf{1}\{y^{(i)} = (k_1, k_2, \ldots, k_p)\} \]

点记法（Punkt-Notation）用于表示边际计数。将某个下标替换为点号，表示对该索引求和：

\[ n_{k_1 \cdot} = \sum_{i=1}^{n} \mathbf{1}\{y_1^{(i)} = k_1\}, \qquad n_{\cdot\cdots\cdot} = \sum_{i=1}^{n} 1 = n \]

1.4 对数线性模型

Log-lineares Modell

Tip乐高积木的比喻 | Die LEGO-Metapher

对数线性模型就像用乐高积木搭建一座建筑。每个变量的”主效应”是基础积木块，而”交互项”是连接不同积木块的特殊连接件。一阶主效应只涉及单个变量，就像独立的柱子；二阶交互项连接两个变量，像横梁连接两根柱子；三阶交互项则让三个变量同时纠缠在一起，像三根柱子交汇处的角钢。完整的”饱和模型”使用了所有可能的连接件，而简约模型则去掉那些不必要的连接件，保留建筑的核心结构。

Das log-lineare Modell baut wie LEGO: Haupteffekte sind Grundbausteine, Interaktionsterme sind Verbindungsstücke. Das saturierte Modell nutzt alle, das sparsamere Modell nur die nötigen.

定义 11.1（对数线性模型）设 \(Y = (Y_1, \ldots, Y_p)\) 为离散值随机变量。若 \(Y\) 的分布满足

\[ P(Y = y; \theta) \propto \exp\left(\theta_0 + \sum_j s_j(y)\theta_j\right) \tag{11.1} \]

其中 \(s_j(\cdot)\) 是从 \(y = (y_1, \ldots, y_p)\) 计算出的若干线性独立统计量，\(\theta = (\theta_0, \theta_1, \ldots)\) 为参数向量，则称该分布遵循对数线性模型。

1.5 二维列联表的具体形式

Konkretes Beispiel: Die zweidimensionale Tafel

对于 \(p = 2\) 的情形，我们定义统计量：

\[ s_{i_1 \cdot}(y) = \mathbf{1}\{y_1 = i_1\}, \quad 1 \le i_1 < K_1 \]

\[ s_{\cdot i_2}(y) = \mathbf{1}\{y_2 = i_2\}, \quad 1 \le i_2 < K_2 \]

\[ s_{i_1 i_2}(y) = \mathbf{1}\{y_1 = i_1, y_2 = i_2\}, \quad 1 \le i_1 < K_1, \; 1 \le i_2 < K_2 \]

注意索引没有遍历所有可能的取值：\(i_1 < K_1\)，\(i_2 < K_2\)。这是为了保证统计量的线性独立性，从而使模型可识别。因为最后一个类别的指示函数可以由前面的导出：

\[ \mathbf{1}\{y_1 = K_1\} = 1 - \sum_{i_1=1}^{K_1-1} s_{i_1 \cdot}(y) \]

对于独立观测 \(Y^{(1)}, \ldots, Y^{(n)}\)，联合概率为：

\[ P(Y^{(1)} = y^{(1)}, \ldots, Y^{(n)} = y^{(n)}) \propto \exp\left\{ n \theta_0 + \sum_{i_1=1}^{K_1-1} n_{i_1 \cdot} \theta_{i_1 \cdot} + \sum_{i_2=1}^{K_2-1} n_{\cdot i_2} \theta_{\cdot i_2} + \sum_{i_1=1}^{K_1-1}\sum_{i_2=1}^{K_2-1} n_{i_1 i_2} \theta_{i_1 i_2} \right\} \]

其中 \(n_{i_1 \cdot} = \sum_{i=1}^{n} \mathbf{1}\{y_1^{(i)} = i_1\}\) 是变量 \(Y_1\) 取值为 \(i_1\) 的观测数，\(n_{i_1 i_2}\) 是二维表的单元格计数。注意 \(n_{i_1 \cdot} = \sum_{i_2=1}^{K_2} n_{i_1 i_2}\)，这就是点记法”对被替换的索引求和”的含义。

将上式简化为 \(K_1 = K_2 = 2\)（二乘二表），仅有一个交互参数 \(\theta_{11}\)：

\[ P(Y^{(1)} = y^{(1)}, \ldots, Y^{(n)} = y^{(n)}) \propto \exp\{\theta_0 + n_{1\cdot}\theta_{1\cdot} + n_{\cdot 1}\theta_{\cdot 1} + n_{11}\theta_{11}\} \]

若令所有涉及双索引的参数为零（即 \(\theta_{11} = 0\)），联合分布可分解为仅依赖 \(Y_1\) 的部分和仅依赖 \(Y_2\) 的部分。此时 \(Y_1\) 与 \(Y_2\) 独立。

1.6 Poisson 模型与多项式模型

Poisson-Modell und Multinomialmodell

对数线性模型本身没有定义一个标准化的概率分布，因为缺少归一化常数。我们可以通过假设单元格计数服从 Poisson 分布来补全：

\[ N_{i_1 i_2} \sim \text{Poisson}(\lambda_{i_1 i_2}) \]

其中

\[ \lambda_{i_1 i_2} = \exp\{\theta_0 + \theta_{i_1 \cdot} + \theta_{\cdot i_2} + \theta_{i_1 i_2}\} \]

并附带边界约束 \(\theta_{K_1 \cdot} = 0\)，\(\theta_{\cdot K_2} = 0\)，\(\theta_{i_1 K_2} = \theta_{K_1 i_2} = 0\)。

对于观测到的单元格计数 \(n_{i_1 i_2}\)，似然函数为：

\[ P(N_{i_1 i_2} = n_{i_1 i_2}) = \prod_{i_1=1}^{K_1}\prod_{i_2=1}^{K_2} \frac{\lambda_{i_1 i_2}^{n_{i_1 i_2}}}{n_{i_1 i_2}!} \exp\{-\lambda_{i_1 i_2}\} \]

\[ \propto \exp\left\{ n_{\cdot\cdot} \theta_0 + \sum_{i_1} n_{i_1 \cdot}\theta_{i_1 \cdot} + \sum_{i_2} n_{\cdot i_2}\theta_{\cdot i_2} + \sum_{i_1}\sum_{i_2} n_{i_1 i_2}\theta_{i_1 i_2} - \sum_{i_1}\sum_{i_2}\lambda_{i_1 i_2}\right\} \]

在 Poisson 模型中，总样本量 \(n_{\cdot\cdot}\) 本身是一个随机变量，这未必合理。如果我们条件在 \(n_{\cdot\cdot} = n\) 上（即固定总观测数），可以证明（参见 Lang, 1996）结果为一个多项式模型：

\[ P(N_{i_1 i_2} = n_{i_1 i_2} \mid n_{\cdot\cdot} = n) = \binom{n_{\cdot\cdot}}{n_{11} \cdots n_{K_1 K_2}} \prod_{i_1=1}^{K_1}\prod_{i_2=1}^{K_2} \pi_{i_1 i_2}^{n_{i_1 i_2}} \]

其中 \(\pi_{i_1 i_2} = \lambda_{i_1 i_2} / \sum_{i_1}\sum_{i_2}\lambda_{i_1 i_2}\)。因此，关于总样本量的信息对对数线性结构没有贡献。

1.7 高维列联表与模型层级

Höherdimensionale Tafeln und Modellhierarchie

对于 \(p > 2\) 的情形，可能的参数集在维度 \(p\) 和类别数方面都急剧增长。一般的 \(K_1 \times K_2 \times \cdots \times K_p\) 维列联表的对数线性模型包含：仅涉及单个变量的主效应项、所有二元配对的交互项、所有三元组的交互项，一直到涉及所有 \(p\) 个变量的最高阶交互项。

以 \(p = 3\)、\(K_1 = K_2 = K_3 = 2\) 为例，这是一个 \(2 \times 2 \times 2\) 的表格，共 8 个单元格。饱和模型（saturiertes Modell）有 8 个参数：\(\theta_0\)，\(\theta_{1\cdot\cdot}\)，\(\theta_{\cdot 1\cdot}\)，\(\theta_{\cdot\cdot 1}\)，\(\theta_{11\cdot}\)，\(\theta_{1\cdot 1}\)，\(\theta_{\cdot 11}\)，\(\theta_{111}\)。如果条件在样本量上，\(\theta_0\) 由该条件决定。

我们用简写记法表示模型。包含所有交互项的完整模型记为 \(1*2*3\)。如果令三阶交互项 \(\theta_{111} = 0\)，则最高阶交互项是二元配对之间的，记为 \(1*2 + 1*3 + 2*3\)。如果进一步令 \(\theta_{11\cdot} = 0\)（去掉 \(Y_1\) 和 \(Y_2\) 之间的二元交互），则记为 \(1*3 + 2*3\)。

1.8 条件独立性与对数线性模型

Bedingte Unabhängigkeit und log-lineare Modelle

我们说 \(Y_1\) 和 \(Y_2\) 在给定 \(Y_3\) 下是条件独立的（bedingt unabhängig），记为 \(Y_1 \perp\!\!\!\perp Y_2 \mid Y_3\)，当且仅当：

\[ P(Y_1 = i_1, Y_2 = i_2 \mid Y_3 = i_3) = P(Y_1 = i_1 \mid Y_3 = i_3) \cdot P(Y_2 = i_2 \mid Y_3 = i_3) \]

对所有 \(1 \le i_1 \le K_1\)，\(1 \le i_2 \le K_2\)，\(1 \le i_3 \le K_3\) 成立。

利用贝叶斯法则可以证明，条件独立意味着联合分布可以分解为两个函数的乘积：

\[ P(Y_1 = i_1, Y_2 = i_2, Y_3 = i_3) \propto h_{13}(i_1, i_3) \cdot h_{23}(i_2, i_3) \]

在对数线性模型中，这等价于：

\[ Y_1 \perp\!\!\!\perp Y_2 \mid Y_3 \quad \Longleftrightarrow \quad \theta_{i_1 i_2 \cdot} = 0 \;\text{ 且 }\; \theta_{i_1 i_2 i_3} = 0 \;\text{ 对所有 } i_1, i_2, i_3 \]

用模型记法表示，这对应于模型 \(1*3 + 2*3\)。条件独立的直观含义是：给定 \(Y_3\) 的取值后，关于 \(Y_2\) 的任何额外信息都不会改变 \(Y_1\) 的分布。

2.1 条件独立性的形式定义

Formale Definition der bedingten Unabhängigkeit

Tip隔音墙的比喻 | Die Schalldämmwand-Metapher

想象三间相邻的房间 A、B、C。C 房间位于 A 和 B 之间。如果 C 房间有完美的隔音墙，那么 A 房间的人听不到 B 房间的声音（反之亦然），即使 A 和 B 各自都能与 C 交流。这就是条件独立性的本质：给定中间变量 C 的信息后，A 和 B 之间不再有额外的信息传递。 图模型中的”缺失边”就像这堵隔音墙 – 它标记了哪些变量之间的信息通道被其他变量”屏蔽”了。

Die bedingte Unabhängigkeit ist wie eine Schalldämmwand: Gegeben die Information in C, fließt keine zusätzliche Information zwischen A und B.

设 \(Y = (Y_1, \ldots, Y_p)\) 为 \(p\) 维随机变量，\(f_{12\cdots p}(y_1, \ldots, y_p)\) 为其联合密度或概率函数。定义三个子集 \(A\)、\(B\)、\(C\)，满足：

\[ A \cap B = A \cap C = B \cap C = \emptyset, \quad A \cup B \cup C \subset \{1, \ldots, p\}, \quad A \neq \emptyset, \; B \neq \emptyset \tag{11.3} \]

我们说 \(A\) 在给定 \(C\) 下与 \(B\) 条件独立，记为 \(A \perp\!\!\!\perp B \mid C\)，当且仅当：

\[ f_{A \cup B \mid C}(y_A, y_B \mid y_C) = f_{A \mid C}(y_A \mid y_C) \cdot f_{B \mid C}(y_B \mid y_C) \]

这等价于联合密度的分解：

\[ f_{A \cup B \cup C}(y_A, y_B, y_C) = h_{A \cup C}(y_A, y_C) \cdot h_{B \cup C}(y_B, y_C) \tag{11.4} \]

2.2 图的定义与缺失边

Graphdefinition und fehlende Kanten

定义图 \(G = (V, E)\)，其中 \(V = \{1, \ldots, p\}\) 是节点集（Knotenmenge），代表变量 \(Y_1, \ldots, Y_p\)；\(E \subset V \times V\) 是边集（Kantenmenge）。图模型的核心思想是：缺失的边才是重点，因为它们代表条件独立性：

\[ i \perp\!\!\!\perp j \mid V \setminus \{i, j\} \quad \Longleftrightarrow \quad (i, j) \notin E \;\text{ 且 }\; (j, i) \notin E \]

换言之，图中缺失一条边等价于这两个变量在给定所有其他变量后是条件独立的。

2.4 三种条件独立性：配对、局部、全局

Drei Arten bedingter Unabhängigkeit: paarweise, lokal, global

配对条件独立性（paarweise bedingte Unabhängigkeit）：如上所述，图中每一条缺失的边对应一个条件独立声明，条件集是所有其他变量。

局部条件独立性（lokale bedingte Unabhängigkeit）：对于节点 \(i\)，令 \(\mathcal{N}(i) = \{j : (i,j) \in E \text{ 或 } (j,i) \in E\}\) 为 \(i\) 的邻居集（Nachbarschaft）。则：

\[ i \perp\!\!\!\perp (V \setminus \{i \cup \mathcal{N}(i)\}) \mid \mathcal{N}(i) \]

例如，在图 11.2 中，变量 2 的邻居是 \(\{1, 3\}\)，因此 \(2 \perp\!\!\!\perp \{4, 5, 6\} \mid \{1, 3\}\)。

全局条件独立性（globale bedingte Unabhängigkeit）：考虑三组变量 \(A\)、\(B\)、\(C\)。如果从 \(A\) 中任一节点到 \(B\) 中任一节点的所有路径都必须经过 \(C\) 中的节点，则 \(A \perp\!\!\!\perp B \mid C\)。

例如，在图 11.2 中，从 \(A = \{4, 5, 6\}\) 到 \(B = \{1, 2\}\) 的所有路径都经过 \(C = \{3\}\)，因此 \(\{1, 2\} \perp\!\!\!\perp \{4, 5, 6\} \mid 3\)。

性质 11.1：给定一个图模型，上述三种条件独立性解读是等价的。证明可参见 Whittaker (2009)、Lauritzen (1996) 或 Maathuis et al. (2018)。

2.7 有向无环图 (DAG)

Gerichtete azyklische Graphen (DAGs)

前面讨论的是无向图（ungerichteter Graph），其中边是对称的。然而，并非所有独立性结构都能用无向图来表达。特别是，可能出现边际独立但条件依赖的情况 – 这需要有向无环图（DAG）来描述。

Tip汇流瓶颈的比喻 | Die Flaschenhals-Metapher

想象两条完全独立的河流（变量 \(Y_1\) 和 \(Y_2\)），它们汇入同一个湖泊（变量 \(Y_3\)）。在正常情况下，两条河流互不影响。但如果你知道了湖泊的水位（条件在 \(Y_3\) 上），那么得知一条河的流量会告诉你另一条河的大致流量（因为总水位是固定的）。这就是”边际独立但条件依赖”的直觉 – DAG 中两个不相连的父节点在条件在它们的共同子节点上时变得依赖。

Zwei unabhängige Flüsse münden in einen See. Kennt man den Wasserstand (bedingt auf \(Y_3\)), so hängen die Flussmengen ab – marginale Unabhängigkeit, aber bedingte Abhängigkeit.

2.8 例 11.2：技术系统故障

Beispiel 11.2: Technisches System

考虑一个由两个独立组件组成的技术系统。\(Y_1\) 表示组件 1 的状态（1 = 故障，0 = 正常），\(Y_2\) 表示组件 2 的状态（类似定义），\(Y_3 = \max(Y_1, Y_2)\) 表示系统状态（1 = 系统故障，0 = 系统正常）。

假设 \(Y_1 \sim B(1, \pi_1)\)，\(Y_2 \sim B(1, \pi_2)\) 且 \(Y_1 \perp\!\!\!\perp Y_2\)。

忽略 \(Y_3\)，利用贝叶斯法则：

\[ P(Y_2 = 1 \mid Y_1 = 1) = \frac{P(Y_1 = 1, Y_2 = 1)}{P(Y_1 = 1)} = \frac{\pi_1 \pi_2}{\pi_1} = \pi_2 \]

所以 \(Y_1\) 和 \(Y_2\) 独立。但条件在 \(Y_3 = 1\) 上：

\[ P(Y_1 = 1, Y_2 = 1 \mid Y_3 = 1) = \frac{\pi_1 \pi_2}{1 - (1-\pi_1)(1-\pi_2)} \]

这不能分解为两个概率的乘积，所以条件在 \(Y_3\) 后 \(Y_1\) 和 \(Y_2\) 不再独立。

更直观地：如果系统故障了（\(Y_3 = 1\)）且组件 1 正常（\(Y_1 = 0\)），则故障一定来自组件 2：

\[ P(Y_2 = 1 \mid Y_3 = 1, Y_1 = 0) = \frac{P(Y_1 = 0, Y_2 = 1, Y_3 = 1)}{P(Y_1 = 0, Y_3 = 1)} = 1 \]

2.9 DAG 的形式定义

Formale Definition eines DAGs

DAG 是一个有向图 \(G = (V, E)\)，其中 \(V = \{1, 2, \ldots, p\}\) 为节点集，\(E \subset V \times V\) 为有向边集。DAG 的两个关键性质是：边不对称（\((i,j) \in E \Rightarrow (j,i) \notin E\)）；无环（不存在 \(v_1, v_2, \ldots, v_k\) 使得 \((v_j, v_{j+1}) \in E\) 且 \(v_1 = v_k\)）。

对于节点 \(i\)，定义其父节点集 \(\text{Pa}(i) = \{j : (j,i) \in E\}\)。DAG 中节点的有序性保证 \(\text{Pa}(i) \subset \{1, \ldots, i-1\}\)。条件独立声明为：

\[ i \perp\!\!\!\perp \{1, \ldots, i-1\} \setminus \text{Pa}(i) \mid \text{Pa}(i) \quad \text{对所有 } i = 1, \ldots, p \]

即：给定 \(Y_i\) 的父节点后，所有”更早”的非父节点都不提供关于 \(Y_i\) 的额外信息。

多元分布可按 DAG 分解为：

\[ f_{Y_1 \cdots Y_p}(y_1, \ldots, y_p) = \prod_{j=1}^{p} f_{Y_j \mid \text{Pa}(j)}(y_j \mid y_{\text{Pa}(j)}) \]

3.1 定义与密度函数

Definition und Dichtefunktion

Tip多元正态的云团比喻 | Die Wolken-Metapher

想象一团漂浮在高维空间中的粒子云。在最简单的情形下（协方差矩阵为单位阵），云团是一个完美的球形，各个方向上的散布相同。当变量之间存在相关性时，这个球被”拉伸”和”旋转”成一个椭球体 – 某些方向上粒子散布更广。协方差矩阵 \(\Sigma\) 精确描述了这个椭球体的形状和朝向。浓度矩阵 \(\Omega = \Sigma^{-1}\) 则提供了另一种视角：它告诉你哪些方向上的变量直接相互影响（而非通过第三个变量的间接传导）。

Die multivariate Normalverteilung ist wie eine Partikelwolke: \(\Sigma\) beschreibt die Form des Ellipsoids, \(\Omega = \Sigma^{-1}\) zeigt die direkten Abhängigkeiten.

定义 11.2：设 \(Y = (Y_1, \ldots, Y_p)\) 为连续随机向量。若 \(Y\) 服从多元正态分布，均值为 \(\mu = (\mu_1, \ldots, \mu_p)^\top\)，\(p \times p\) 维协方差矩阵为 \(\Sigma\)，则其密度函数为：

\[ f(y; \mu, \Sigma) = \frac{1}{(2\pi)^{p/2} |\Sigma|^{1/2}} \exp\left\{-\frac{1}{2}(y - \mu)^\top \Sigma^{-1}(y - \mu)\right\} \tag{11.5} \]

其中假设 \(\Sigma\) 正定，即对所有非零向量 \(a \in \mathbb{R}^p\) 有 \(a^\top \Sigma a > 0\)。

3.2 最大似然估计

给定 i.i.d. 数据 \(y^{(i)}\)，\(i = 1, \ldots, n\)，最大似然估计为：

\[ \hat{\mu} = \frac{1}{n}\sum_{i=1}^{n} y^{(i)}, \qquad \hat{\Sigma} = \frac{1}{n}\sum_{i=1}^{n} (y^{(i)} - \hat{\mu})(y^{(i)} - \hat{\mu})^\top \]

3.3 浓度矩阵与对数线性结构

Konzentrationsmatrix und log-lineare Struktur

令 \(\Omega = \Sigma^{-1}\) 为浓度矩阵（Konzentrationsmatrix，又称精度矩阵 / Präzisionsmatrix），\(\Omega_{jl}\) 为其 \((j, l)\) 元素。将密度函数 (11.5) 中的指数部分展开：

\[ f(y; \mu, \Sigma) \propto \exp\left\{-\frac{1}{2}\sum_{j=1}^{p}\sum_{l=1}^{p} \Omega_{jl}(y_j - \mu_j)(y_l - \mu_l)\right\} \]

进一步化简：

\[ \propto \exp\left\{-\sum_{j=1}^{p}\sum_{l>j} \Omega_{jl} y_j y_l + \sum_{j=1}^{p}\left[-\frac{1}{2}\Omega_{jj} y_j^2 + \left(\sum_{l=1}^{p}\Omega_{jl}\mu_l\right)y_j\right]\right\} \tag{11.6} \]

定义统计量 \(s_j(y) = (y_j^2, y_j)^\top\) 和 \(s_{jl}(y) = y_j y_l\)，参数 \(\theta_j = (-\frac{1}{2}\Omega_{jj}, \sum_l \Omega_{jl}\mu_l)^\top\) 和 \(\theta_{jl} = -\Omega_{jl}\)，则 (11.6) 可以重写为：

\[ f(y; \mu, \Sigma) \propto \exp\left\{\sum_{j=1}^{p}\sum_{l>j} \theta_{lj} s_{lj}(y) + \sum_{j=1}^{p} \theta_j^\top s_j(y)\right\} \tag{11.7} \]

这揭示了两个极其重要的性质。

性质一：与离散变量的对数线性模型不同，多元正态分布不包含高阶交互项。它只有二阶（配对）交互项。这意味着多元正态分布的依赖结构完全由配对关系决定。

性质二：条件独立性可以直接从浓度矩阵中读取：

\[ \Omega_{jl} = 0 \quad \Longleftrightarrow \quad Y_j \perp\!\!\!\perp Y_l \mid Y_1, \ldots, Y_p \setminus \{j, l\} \tag{11.8} \]

这是一个非常优雅的结论：浓度矩阵中的零元素恰好对应图模型中的缺失边。

Tip协方差 vs 浓度矩阵的比喻 | Kovarianz- vs. Konzentrationsmatrix

想象一个社交网络。协方差矩阵 \(\Sigma\) 就像衡量两个人之间的”总相关性” – 这包括通过共同朋友的间接关联。如果 Alice 和 Bob 都与 Charlie 关系很好，那么即使 Alice 和 Bob 从未直接交流过，他们之间的协方差可能也不为零（因为共同的朋友 Charlie 制造了统计关联）。

浓度矩阵 \(\Omega = \Sigma^{-1}\) 则过滤掉了这些间接效应。\(\Omega_{jl} = 0\) 意味着在已知所有其他人的信息后，Alice 和 Bob 之间没有直接联系。这就是为什么浓度矩阵对图模型如此重要 – 它揭示的是直接的条件依赖关系。

\(\Sigma\) misst die Gesamtkorrelation (inklusive indirekter Pfade). \(\Omega = \Sigma^{-1}\) zeigt nur die direkten bedingten Abhängigkeiten.

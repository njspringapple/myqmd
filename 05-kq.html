最小二乘法：从直觉到算法

1 前置知识回顾

在开始之前，确认你对以下概念有基本了解：

  • 矩阵范数与条件数：\(\kappa(\mathbf{A}) = \|\mathbf{A}\|\|\mathbf{A}^{-1}\|\)，条件数越大问题越”病态”
  • 正交矩阵：满足 \(\mathbf{Q}^T\mathbf{Q} = \mathbf{I}\)，保持向量长度和角度不变
  • 矩阵分解：LU分解、Cholesky分解、SVD（奇异值分解）
  • 误差分析：绝对误差、相对误差、误差分解（算法误差 + 输入传播误差）

如果对条件数和误差分析还不熟悉，建议先回顾上一章。本章将从”当方程组无精确解时怎么办”这个最实际的问题出发，系统建立最小二乘法的理论和算法框架。

2 动机：为什么需要最小二乘？

2.1 一个简单的例子

假设你是一名房产中介，想根据房屋面积预测房价。你手上有 5 套房子的数据：

# 房屋数据
area <- c(50, 70, 80, 100, 120)    # 面积 (m²)
price <- c(150, 200, 230, 310, 350)  # 价格 (千€)

cat("面积 (m²):  ", area, "\n")
面积 (m²):   50 70 80 100 120 
cat("价格 (千€): ", price, "\n")
价格 (千€):  150 200 230 310 350

你想找一个线性模型 \(y = \beta_0 + \beta_1 x\) 来描述”面积→价格”的关系。把每个数据点代入，得到 5 个方程、2 个未知数：

\[ \begin{cases} \beta_0 + 50\beta_1 = 150 \\ \beta_0 + 70\beta_1 = 200 \\ \beta_0 + 80\beta_1 = 230 \\ \beta_0 + 100\beta_1 = 310 \\ \beta_0 + 120\beta_1 = 350 \end{cases} \]

5 个方程，2 个未知数——方程比未知数多，这叫超定系统（overdetermined system）。一般来说，不可能找到一组 \((\beta_0, \beta_1)\) 同时满足所有方程。

# 可视化：数据点不在一条直线上

     xlab = "面积 (m²)", ylab = "价格 (千€)",
     main = "房价数据：无法找到一条直线精确穿过所有点",
     xlim = c(40, 130), ylim = c(100, 400))
grid()

legend("topleft", "数据点", pch = 19, col = "steelblue")

问题：既然没有精确解，能不能找一个”最好的近似”？

答案：最小二乘法——找使所有偏差的平方和最小的那组参数。

2.2 写成矩阵形式

把上面的方程组写成 \(\mathbf{A}\boldsymbol{\beta} = \mathbf{b}\)：

\[ \underbrace{\begin{pmatrix} 1 & 50 \\ 1 & 70 \\ 1 & 80 \\ 1 & 100 \\ 1 & 120 \end{pmatrix}}_{\mathbf{A}} \underbrace{\begin{pmatrix} \beta_0 \\ \beta_1 \end{pmatrix}}_{\boldsymbol{\beta}} = \underbrace{\begin{pmatrix} 150 \\ 200 \\ 230 \\ 310 \\ 350 \end{pmatrix}}_{\mathbf{b}} \]

最小二乘法就是要找 \(\hat{\boldsymbol{\beta}}\) 使得 \(\|\mathbf{A}\hat{\boldsymbol{\beta}} - \mathbf{b}\|_2\) 最小。

2.3 更广泛的应用

最小二乘不仅仅是”拟合直线”。它是统计学和机器学习中最基础的工具之一，应用场景包括广义线性模型和加性模型（通过迭代最小二乘求近似解）、深度学习（最小化平方误差损失函数）、计算机视觉（相机标定、3D重建）、信号处理（降噪、信号估计）以及大地测量（GPS定位）等。

3 最小二乘问题的形式化

3.1 问题定义

最小二乘问题（KQ问题）：给定矩阵 \(\mathbf{A} \in \mathbb{R}^{m \times n}\) 和向量 \(\mathbf{b} \in \mathbb{R}^m\)，寻找

\[ \hat{\mathbf{x}} = \arg\min_{\mathbf{x} \in \mathbb{R}^n} \|\mathbf{A}\mathbf{x} - \mathbf{b}\|_2 \]

注意两种特殊情况：如果 \(\mathbf{Ax} = \mathbf{b}\) 恰好有精确解 \(\mathbf{x}^*\)，那么 \(\|\mathbf{Ax}^* - \mathbf{b}\| = 0\)，此时精确解也是最小二乘解。最小二乘方法特别适用于没有精确解的超定系统 \(\mathbf{Ax} \approx \mathbf{b}\)（\(m > n\)）。

备注：矩阵 A(mxn)，其中 m > n, 即 有 n 个变量，m 个方程，此时，方程通常无解（除非 Ax = b 这个 b 正好落在 A 的列空间）。由列空间 1到n 维的一个空间， 构成一个“平面”，在 m 维 空间上向量 b 在列空间的投影是残差最小的近似解。这种 m > n 的矩阵也叫“超定”系统。

通常假设 A 的列空间 “满秩”，即 有n个线性无关向量或者有n个非零奇异值

核心几何事实：

  • \(\text{col}(\mathbf{A})\)：矩阵 \(\mathbf{A}\) 的列空间——所有可能的 \(\mathbf{Ax}\) 形成的子空间
  • \(\hat{\mathbf{b}} = \mathbf{A}\hat{\mathbf{x}}\)：\(\mathbf{b}\) 在列空间上的正交投影
  • \(\mathbf{r} = \mathbf{b} - \hat{\mathbf{b}}\)：残差向量，垂直于列空间

解释：

  1. 想象矩阵 A 的每一列都是一个方向箭头，列空间就是：用这些方向箭头任意组合（加权相加）能到达的所有位置。
  2. Ax 的意思就是：用向量 x 的各个分量去加权组合 A 的列
  3. 现在你有一个目标向量 b，但它可能不在列空间里
  4. 正交投影就是：在列空间内找到离 b 最近的点 \(\hat{b}\)
  5. 具体做法是：从 b 向列空间垂直投下一条影子，影子落点就是 \(\hat{b}\)

想象下，如果2列，且线性无关，那列空间就是整个二维空间，b 一定在空间内。

如果A是3x2，即3个方程只有2个变量（列 < 行），A 的列空间是三维空间内的2维平面，b 就可能不在这个平面，

因此需要找到投影 \(\hat{b}\) ，使得 \(|b - A\hat{x}|\) 最小

同理，如果列为2，但是秩为1，列空间就是直线，大部分b不在直线上，投影才有意义。

投影公式：\(\hat{b} = A\hat{x}\)

  • b 待投影的向量
  • \(\hat{b}\) 投影后的向量（结果）
  • \(A\) 矩阵
  • \(\hat{x}\) 投影系数（待求）

直接求解：

  • \(\hat{x} = (A^TA)^{-1}A^Tb\)
  • \(\hat{b} = A(A^TA)^{-1}A^Tb\)

简单理解：\(A^T\) 用于一次性检查 残差 r 是否与 A 的列空间（所有列）垂直

为什么投影给出最小距离？ 这是初等几何的事实：从空间中一个点到一个平面（或直线），垂线段最短。

# 为什么投影是最优的？画出不同的 x 对应的 ||Ax - b||

定理：最小二乘问题的解集等于线性方程组 \(\mathbf{A}\hat{\mathbf{x}} = \hat{\mathbf{b}}\) 的解集，其中 \(\hat{\mathbf{b}} = \text{proj}_{\text{col}(\mathbf{A})}\mathbf{b}\)。

4 正规方程

4.1 推导

由几何解释，残差 \(\mathbf{r} = \mathbf{b} - \mathbf{A}\hat{\mathbf{x}}\) 垂直于列空间，即对 \(\mathbf{A}\) 的每一列 \(\mathbf{a}_j\) 都有 \(\mathbf{a}_j^T \mathbf{r} = 0\)。写成矩阵形式：

\[ \mathbf{A}^T(\mathbf{b} - \mathbf{A}\hat{\mathbf{x}}) = \mathbf{0} \]

展开得到正规方程：

\[ \mathbf{A}^T\mathbf{A}\hat{\mathbf{x}} = \mathbf{A}^T\mathbf{b} \]

另一种推导（微积分方法）：令 \(\phi(\mathbf{x}) = \|\mathbf{Ax} - \mathbf{b}\|^2\)，展开后对 \(\mathbf{x}\) 求导令其为零，也得到同样的方程。

4.2 唯一性条件

定理：如果 \(\text{rank}(\mathbf{A}) = n\)（即 \(\mathbf{A}\) 的列满秩），则 \(\mathbf{A}^T\mathbf{A}\) 可逆，最小二乘问题有唯一解：

\[ \hat{\mathbf{x}} = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\mathbf{b} =: \mathbf{A}^+\mathbf{b} \]

这里 \(\mathbf{A}^+ = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\) 称为 \(\mathbf{A}\) 的 Moore-Penrose 伪逆。

  • 对称性：P^T = P

5 条件数：最小二乘有多敏感？

5.1 为什么要关心条件数？

上一章我们学过：条件数衡量问题对输入扰动的敏感程度。对于最小二乘问题，输入是 \(\mathbf{A}\) 和 \(\mathbf{b}\)，输出是 \(\hat{\mathbf{x}}\)。问题是：\(\mathbf{A}\) 或 \(\mathbf{b}\) 的微小变化，会导致 \(\hat{\mathbf{x}}\) 偏多少？

对于非方阵 \(\mathbf{A} \in \mathbb{R}^{m \times n}\)，条件数定义为：

\[ \kappa_2(\mathbf{A}) = \|\mathbf{A}\|_2 \|\mathbf{A}^+\|_2 = \frac{\sigma_{\max}(\mathbf{A})}{\sigma_{\min}(\mathbf{A})} \]

当 \(\text{rank}(\mathbf{A}) < n\) 时，\(\kappa(\mathbf{A}) = \infty\)。

5.2 关于 b 的条件数

固定 \(\mathbf{A}\)，看 \(\mathbf{b}\) 的扰动如何影响解。

定理：设 \(f(\mathbf{b}) = \mathbf{A}^+\mathbf{b}\)，则

\[ \frac{\|f(\tilde{\mathbf{b}}) - f(\mathbf{b})\|}{\|f(\mathbf{b})\|} \leq \kappa \cdot \frac{\|\tilde{\mathbf{b}} - \mathbf{b}\|}{\|\mathbf{b}\|}, \quad \text{其中} \quad \kappa = \kappa_2(\mathbf{A}) \cdot \frac{\|\mathbf{b}\|}{\|\text{proj}_{\text{col}(\mathbf{A})}\mathbf{b}\|} \]

证明思路：

\[ \frac{\|\mathbf{A}^+\tilde{\mathbf{b}} - \mathbf{A}^+\mathbf{b}\|}{\|\mathbf{A}^+\mathbf{b}\|} = \frac{\|\mathbf{A}^+(\tilde{\mathbf{b}} - \mathbf{b})\|}{\|\mathbf{A}^+\mathbf{b}\|} \leq \frac{\|\mathbf{A}^+\|\|\tilde{\mathbf{b}} - \mathbf{b}\|}{\|\mathbf{A}^+\mathbf{b}\|} \]

利用 \(\|\mathbf{A}^+\mathbf{b}\| \geq \|\mathbf{A}\mathbf{A}^+\mathbf{b}\|/\|\mathbf{A}\| = \|\text{proj}_{\text{col}(\mathbf{A})}\mathbf{b}\|/\|\mathbf{A}\|\)，即得。

  • 最坏情况：\(\mathbf{b} \perp \text{col}(\mathbf{A})\)，此时投影 = 0，\(\kappa = \infty\)

直观来说：如果 \(\mathbf{b}\) 几乎垂直于列空间，那么 \(\hat{\mathbf{b}}\) 几乎为零，微小扰动就能让投影方向剧烈变化。

5.3 关于 A 的条件数

固定 \(\mathbf{b}\)，看 \(\mathbf{A}\) 的扰动如何影响解。

定理：设 \(f(\mathbf{A}) = \mathbf{A}^+\mathbf{b}\)，则

\[ \frac{\|f(\tilde{\mathbf{A}}) - f(\mathbf{A})\|}{\|f(\mathbf{A})\|} \leq \kappa \cdot \frac{\|\tilde{\mathbf{A}} - \mathbf{A}\|}{\|\mathbf{A}\|}, \quad \text{其中} \quad \kappa = \kappa_2(\mathbf{A}) + \kappa_2(\mathbf{A})^2 \cdot \frac{\|\mathbf{r}\|}{\|\text{proj}_{\text{col}(\mathbf{A})}\mathbf{b}\|} \]

其中 \(\mathbf{r} = \mathbf{b} - \mathbf{A}\hat{\mathbf{x}}\) 是残差向量。

6.1 从正规方程说起

正规方程 \(\mathbf{A}^T\mathbf{A}\hat{\mathbf{x}} = \mathbf{A}^T\mathbf{b}\) 给出了最直接的求解思路：先算 \(\mathbf{A}^T\mathbf{A}\)，然后求解线性方程组。

但是——先看一个”错误”的实现：

6.2 方法 1：正规方程 + Cholesky 分解

由于 \(\mathbf{A}^T\mathbf{A}\) 是对称正定矩阵（当 \(\mathbf{A}\) 列满秩时），可以使用 Cholesky 分解。

算法步骤：

  1. 计算 \(\mathbf{M} = \mathbf{A}^T\mathbf{A}\) 和 \(\mathbf{c} = \mathbf{A}^T\mathbf{b}\)
  2. Cholesky 分解：\(\mathbf{M} = \mathbf{LL}^T\)
  3. 前向替换：求解 \(\mathbf{Ly} = \mathbf{c}\)
  4. 回代：求解 \(\mathbf{L}^T\hat{\mathbf{x}} = \mathbf{y}\)
# Cholesky 方法
kq_solve_chol <- function(A, b) {
  M <- crossprod(A)       # = A^T A（比 t(A) %*% A 更高效）
  c_vec <- crossprod(A, b)  # = A^T b
  L <- chol(M)            # R 的 chol() 返回上三角 R，满足 M = R^T R
  # 求解 R^T R x = c，即先解 R^T y = c，再解 R x = y
  y <- forwardsolve(t(L), c_vec)
  x <- backsolve(L, y)
  x
}

6.4 方法 2：QR 分解

6.4.1 QR 分解是什么？

定义：对于 \(\mathbf{A} \in \mathbb{R}^{m \times n}\)（\(m \geq n\)），存在正交矩阵 \(\mathbf{Q} \in \mathbb{R}^{m \times m}\) 使得

\[ \mathbf{Q}^T\mathbf{A} = \begin{pmatrix} \mathbf{R} \\ \mathbf{0}_{(m-n) \times n} \end{pmatrix} \]

其中 \(\mathbf{R} \in \mathbb{R}^{n \times n}\) 是上三角矩阵。等价地写成

\[ \mathbf{A} = \mathbf{Q} \begin{pmatrix} \mathbf{R} \\ \mathbf{0} \end{pmatrix} \]

6.4.2 QR 分解如何求解最小二乘？

定理：设 \(\mathbf{Q}^T\mathbf{A} = \begin{pmatrix} \mathbf{R} \\ \mathbf{0} \end{pmatrix}\)，\(\mathbf{Q}^T\mathbf{b} = \begin{pmatrix} \mathbf{c}_1 \\ \mathbf{c}_2 \end{pmatrix}\)，其中 \(\mathbf{R}\) 可逆，则 \(\hat{\mathbf{x}} = \mathbf{R}^{-1}\mathbf{c}_1\)。

证明——这是全文最优美的推导之一：

\[ \|\mathbf{Ax} - \mathbf{b}\|^2 = \|\mathbf{Q}^T(\mathbf{Ax} - \mathbf{b})\|^2 = \|\mathbf{Q}^T\mathbf{Ax} - \mathbf{Q}^T\mathbf{b}\|^2 \]

\[ = \left\| \begin{pmatrix} \mathbf{R} \\ \mathbf{0} \end{pmatrix} \mathbf{x} - \begin{pmatrix} \mathbf{c}_1 \\ \mathbf{c}_2 \end{pmatrix} \right\|^2 = \|\mathbf{Rx} - \mathbf{c}_1\|^2 + \|\mathbf{c}_2\|^2 \]

第二项 \(\|\mathbf{c}_2\|^2\) 不依赖于 \(\mathbf{x}\)，所以最小化等价于令 \(\|\mathbf{Rx} - \mathbf{c}_1\| = 0\)，即 \(\hat{\mathbf{x}} = \mathbf{R}^{-1}\mathbf{c}_1\)。

而且 \(\|\mathbf{c}_2\|^2\) 就是最小残差的平方！

# QR 方法求解最小二乘
kq_solve_qr <- function(A, b) {
  qr_dec <- qr(A)
  Q <- qr.Q(qr_dec, complete = TRUE)
  R <- qr.R(qr_dec)

  n <- ncol(A)
  c1 <- Qtb[1:n]
  c2 <- Qtb[(n+1):length(Qtb)]

  x_hat <- backsolve(R, c1)

  list(x = x_hat,
       min_residual_norm = vnorm(c2),
       c1 = c1, c2 = c2)
}

6.4.3 构造 Q：Householder 反射

QR 分解的关键步骤是构造正交矩阵 \(\mathbf{Q}\)。最常用的方法是 Householder 反射。

思想：通过一系列反射操作，逐列消去 \(\mathbf{A}\) 的下三角元素。每次反射由一个向量 \(\mathbf{v}\) 定义：

\[ \mathbf{H} = \mathbf{I} - 2\frac{\mathbf{v}\mathbf{v}^T}{\mathbf{v}^T\mathbf{v}} \]

\(\mathbf{H}\) 是正交的、对称的，且 \(\mathbf{H}^2 = \mathbf{I}\)（反射两次回到原处）。

6.5 方法 3：SVD（奇异值分解）

6.5.1 用 SVD 求解最小二乘

设 \(\mathbf{A} = \mathbf{U\Sigma V}^T\) 是 \(\mathbf{A}\) 的 SVD，则

\[ \hat{\mathbf{x}} = \mathbf{A}^+\mathbf{b} = \mathbf{V}\boldsymbol{\Sigma}^+\mathbf{U}^T\mathbf{b} \]

其中 \(\boldsymbol{\Sigma}^+\) 是将 \(\boldsymbol{\Sigma}\) 的非零对角元素取倒数得到的矩阵。

# SVD 方法
kq_solve_svd <- function(A, b) {
  sv <- svd(A)
  # A⁺ = V Σ⁺ U^T
  # x = V Σ⁺ U^T b

  # 计算 U^T b
  Utb <- t(sv$u) %*% b

  # 除以奇异值（对应 Σ⁺ 的操作）
  # 只取前 n 个分量（对应非零奇异值）
  n <- length(sv$d)
  coeffs <- Utb[1:n] / sv$d

  # 乘以 V
  x_hat <- sv$v %*% coeffs

  list(x = x_hat, singular_values = sv$d)
}

9 总结

9.1 知识地图

最小二乘法
├── 问题定义
│   ├── min ||Ax - b||₂（超定系统的最佳近似）
│   ├── 几何本质：将 b 投影到 col(A)
│   └── 正规方程：A^T A x̂ = A^T b
│
├── 条件数分析
│   ├── 关于 b：κ₂(A) · ||b|| / ||proj b||
│   ├── 关于 A：κ₂(A) + κ₂(A)² · ||r|| / ||proj b||
│   ├── 精确解时：κ₂(A)（与方程组相同）
│   └── 一般情况：O(κ₂(A)²)（条件数平方化！）
│
├── 三种求解算法
│   ├── Cholesky（正规方程）
│   │   ├── 最快
│   │   └── 风险：κ² 平方化
│   ├── QR 分解 ← 首选
│   │   ├── 利用正交不变性
│   │   ├── Householder 反射构造 Q
│   │   └── 条件数不平方化
│   └── SVD
│       ├── 最稳定，最慢
│       ├── 处理秩亏矩阵
│       └── 给出最小范数解
│
└── 实际联系
    ├── R 的 lm() 使用 QR 分解
    ├── 多项式拟合 → Vandermonde 矩阵条件数指数增长
    └── 岭回归 → 降低条件数，稳定参数估计

9.2 核心公式卡

概念 公式
最小二乘问题 \(\hat{\mathbf{x}} = \arg\min_{\mathbf{x}} \|\mathbf{Ax} - \mathbf{b}\|_2\)
正规方程 \(\mathbf{A}^T\mathbf{A}\hat{\mathbf{x}} = \mathbf{A}^T\mathbf{b}\)
伪逆 \(\mathbf{A}^+ = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\)
条件数平方化 \(\kappa(\mathbf{A}^T\mathbf{A}) = \kappa(\mathbf{A})^2\)
QR 分解求解 \(\hat{\mathbf{x}} = \mathbf{R}^{-1}\mathbf{c}_1\)，其中 \(\mathbf{Q}^T\mathbf{b} = (\mathbf{c}_1, \mathbf{c}_2)^T\)
SVD 求解 \(\hat{\mathbf{x}} = \mathbf{V}\boldsymbol{\Sigma}^+\mathbf{U}^T\mathbf{b}\)

9.3 实践指南

你遇到了什么 怎么做
一般线性回归 用 lm() 或 qr.solve()（QR 分解）
特征共线性 用岭回归或 SVD，检查 \(\kappa(\mathbf{X}^T\mathbf{X})\)
高阶多项式拟合不稳定 降低阶数，或用正交多项式基
需要最快速度且 \(\mathbf{A}\) 良态 用 Cholesky
\(\mathbf{A}\) 可能秩亏 用 SVD，检查奇异值分布
solve() 报错 检查 \(\kappa(\mathbf{A})\)，考虑正则化

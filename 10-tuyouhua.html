1 凸组合：一切的起点

Konvexe Kombination: Der Ausgangspunkt von allem

1.1 从线性组合到凸组合

Von der Linearkombination zur konvexen Kombination

你已经熟悉了线性组合的概念：给定向量 \(\mathbf{x}_1, \ldots, \mathbf{x}_k\)，任意系数 \(w_1, \ldots, w_k \in \mathbb{R}\) 下的组合 \(\sum_{i=1}^k w_i \mathbf{x}_i\) 就是一个线性组合。

现在我们加上两个限制条件，就得到了凸组合（konvexe Kombination / convex combination）：

\[ \mathbf{x} = \sum_{i=1}^{k} w_i \mathbf{x}_i, \quad \text{其中 } w_i \geq 0 \;\;\text{且}\;\; \sum_{i=1}^{k} w_i = 1 \]

这两个限制意味着什么？所有权重非负且总和为 1——这不就是加权平均吗！

想想离散随机变量 \(X\) 的期望值 \(\mathbb{E}(X) = \sum_{x_j \in T_X} \mathbb{P}(X = x_j) \, x_j\)，其中每个概率 \(\mathbb{P}(X = x_j) \geq 0\) 且所有概率之和为 1。期望值就是支撑集中各取值的凸组合。

1.2 两点的凸组合 = 线段

Konvexe Kombination zweier Punkte = Strecke

最简单的情况：两个向量 \(\mathbf{x}_1, \mathbf{x}_2\) 的所有凸组合构成什么？

\[ \mathbf{x} = w \,\mathbf{x}_1 + (1-w)\, \mathbf{x}_2, \quad w \in [0, 1] \]

当 \(w\) 从 0 连续变到 1 时，\(\mathbf{x}\) 从 \(\mathbf{x}_2\) “滑动”到 \(\mathbf{x}_1\)——这正是连接两点的线段！

1.3 凸包：所有凸组合的集合

Konvexe Hülle: Die Menge aller konvexen Kombinationen

给定一组点 \(\mathcal{X} = \{\mathbf{x}_1, \ldots, \mathbf{x}_k\}\)，它们所有可能的凸组合构成的集合叫做凸包（konvexe Hülle / convex hull）：

\[ \operatorname{conv}(\mathcal{X}) = \left\{\sum_{i=1}^{N} w_i \mathbf{x}_i \;:\; N \in \mathbb{N},\; \mathbf{x}_i \in \mathcal{X},\; w_i \geq 0,\; \sum_{i=1}^{N} w_i = 1\right\} \]

那些只能通过某个 \(w_i = 1\)（其余权重为 0）才能到达的点称为极值点（Extrempunkt）——它们是凸包的”骨架”。

Important💡 关键洞察 | Schlüsselerkenntnis

凸组合 = 加权平均。凸包是包含一组点的最小凸集，它的”角”就是极值点。

Konvexe Kombination = gewichteter Durchschnitt. Die konvexe Hülle ist die kleinste konvexe Menge, die alle Punkte enthält.

凸包还有一个等价定义：它是包含 \(\mathcal{X}\) 的所有凸集的交集：\(\operatorname{conv}(\mathcal{X}) = \bigcap_{\mathcal{X} \subseteq \mathcal{Y},\, \mathcal{Y}\text{ 凸}} \mathcal{Y}\)。

2.1 定义：线段检验

Definition: Der Streckentest

一个集合 \(\mathcal{X} \subset V\)（向量空间 \(V\) 的子集）是凸集（konvexe Menge / convex set），当且仅当对所有 \(\mathbf{x}, \mathbf{y} \in \mathcal{X}\) 及所有 \(\lambda \in [0, 1]\)，有

\[ \lambda \mathbf{x} + (1 - \lambda)\mathbf{y} \in \mathcal{X} \]

用一句话概括：任取集合中的两个点，连接它们的线段完全落在集合内部。换言之，凸集没有”凹陷”或”孔洞”。

2.2 小测验：哪些集合是凸的？

Quiz: Welche Mengen sind konvex?

Tip🤔 想一想 | Denken Sie nach

判断以下集合是否为凸集：

  1. 空集 \(\emptyset\)
  2. 整个向量空间 \(V\)
  3. 单位球面 \(\{\mathbf{x} \in V : \|\mathbf{x}\| = 1\}\)
  4. 闭球 \(\{\mathbf{x} \in V : \|\mathbf{x}\| \leq r\}\)
  5. 集合 \(\{(x, y) \in \mathbb{R}^2 : y \leq x^2\}\)
Tip🔍 答案（点击展开）
  1. 凸 ✅ — 空集平凡地满足定义（没有两个点可以选）
  2. 凸 ✅ — \(V\) 对线性组合封闭，当然对凸组合也封闭
  3. 不凸 ❌ — 取球面上两个对径点 \(\mathbf{x}\) 和 \(-\mathbf{x}\)，它们的中点 \(\mathbf{0}\) 不在球面上
  4. 凸 ✅ — 由三角不等式可证
  5. 不凸 ❌ — 取 \((-1, 1)\) 和 \((1, 1)\)（都满足 \(y \leq x^2\)），它们的中点 \((0, 1)\) 不满足 \(1 \leq 0^2\)

2.3 重要凸集实例

Wichtige Beispiele konvexer Mengen

2.3.1 闭球与开球

闭球 \(\{\mathbf{x} : \|\mathbf{x}\| \leq r\}\) 和开球 \(\{\mathbf{x} : \|\mathbf{x}\| < r\}\) 都是凸集。直觉很清晰：球没有任何”凹陷”。

2.3.2 半空间

\(\{\mathbf{x} \in \mathbb{R}^n : \mathbf{a}^\top \mathbf{x} \leq b\}\) 是凸集。多个半空间的交集形成多面体（Polytop / Polyhedron）——这对线性规划至关重要。

2.3.3 单纯形 \(\Delta^N\)

\[ \Delta^N = \left\{\mathbf{x} \in \mathbb{R}^N : x_i \geq 0,\; \sum_{i=1}^{N} x_i = 1\right\} \]

单纯形就是”所有概率分布”的集合！它的极值点是标准基向量 \(\mathbf{e}_1, \ldots, \mathbf{e}_N\)（对应”确定性”的概率分布，即某事件概率为 1）。

2.3.4 半正定矩阵集合

\(\mathbb{R}^{n \times n}\) 中所有半正定矩阵（PSD, \(\mathbf{A} \succeq 0\)，即 \(\mathbf{x}^\top \mathbf{A} \mathbf{x} \geq 0\) 对所有 \(\mathbf{x}\) 成立）的集合是凸集。

Tip🔍 证明（点击展开）

设 \(\mathbf{A}_1, \mathbf{A}_2 \succeq 0\)，\(\lambda \in [0, 1]\)。令 \(\mathbf{A}_\lambda = \lambda \mathbf{A}_1 + (1 - \lambda)\mathbf{A}_2\)。

对任意 \(\mathbf{x} \in \mathbb{R}^n\)：

\[ \mathbf{x}^\top \mathbf{A}_\lambda \mathbf{x} = \lambda \underbrace{\mathbf{x}^\top \mathbf{A}_1 \mathbf{x}}_{\geq 0} + (1 - \lambda)\underbrace{\mathbf{x}^\top \mathbf{A}_2 \mathbf{x}}_{\geq 0} \geq 0 \quad \square \]

应用：协方差矩阵是对称半正定的，所以所有合法的协方差矩阵构成一个凸集。

2.4 凸性在运算下的保持

Erhaltung der Konvexität unter Operationen

许多重要运算都保持凸性——这使得我们可以通过简单集合”搭建”复杂的凸集。

Important💡 凸性保持定理 | Erhaltungssatz der Konvexität

设 \(\mathcal{X}, \mathcal{X}_1, \mathcal{X}_2, \ldots\) 为凸集，则以下集合也是凸的：

  • 交集：\(\mathcal{S} = \bigcap_{i=1}^{\infty} \mathcal{X}_i\)
  • 笛卡尔积：\(\mathcal{X}_1 \times \cdots \times \mathcal{X}_k\)
  • 线性映射的像：若 \(f(\mathbf{x}) = \mathbf{A}\mathbf{x} + \mathbf{b}\)，则 \(f(\mathcal{X})\) 是凸的
  • Minkowski 和：\(\mathcal{X}_1 \oplus \mathcal{X}_2 := \{\mathbf{x}_1 + \mathbf{x}_2 : \mathbf{x}_1 \in \mathcal{X}_1,\, \mathbf{x}_2 \in \mathcal{X}_2\}\)
Tip🔍 证明概要（点击展开）

交集：设 \(\mathbf{x}, \mathbf{y} \in \mathcal{S}\)，则 \(\mathbf{x}, \mathbf{y}\) 属于每个 \(\mathcal{X}_i\)。每个 \(\mathcal{X}_i\) 是凸的，所以 \(\lambda \mathbf{x} + (1-\lambda)\mathbf{y} \in \mathcal{X}_i\) 对所有 \(i\) 成立，从而它属于 \(\mathcal{S}\)。

笛卡尔积：逐分量使用凸性即可。

线性映射：设 \(\mathbf{y}_1 = f(\mathbf{x}_1), \mathbf{y}_2 = f(\mathbf{x}_2) \in f(\mathcal{X})\)，由 \(f\) 的线性性得 \(\lambda \mathbf{y}_1 + (1-\lambda)\mathbf{y}_2 = f(\lambda\mathbf{x}_1 + (1-\lambda)\mathbf{x}_2) \in f(\mathcal{X})\)。

Minkowski 和：由笛卡尔积的凸性加上求和映射的线性性得出。

一个重要的应用：多面体是半空间的交集。每个半空间是凸的，交集保持凸性，所以多面体是凸的——这是线性规划的理论基础。

3.1 定义：弦不低于弧

Definition: Die Sehne liegt nie unter dem Bogen

设 \(\mathcal{X}\) 是凸集，\(f : \mathcal{X} \to \mathbb{R}\)。若对所有 \(\mathbf{x}, \mathbf{y} \in \mathcal{X}\) 及所有 \(\lambda \in [0, 1]\)，有

\[ f\bigl(\lambda \mathbf{x} + (1 - \lambda)\mathbf{y}\bigr) \leq \lambda\, f(\mathbf{x}) + (1 - \lambda)\, f(\mathbf{y}) \]

则称 \(f\) 是凸函数（konvexe Funktion / convex function）。

直觉：连接函数图像上任意两点的弦（右边）永远不会低于函数的弧（左边）。凸函数是”向上开口”的。若 \(-f\) 是凸的，则称 \(f\) 是凹函数（konkave Funktion）。

3.5 重要的多元凸函数

Wichtige multivariate konvexe Funktionen

以下多元函数都是凸的：

仿射函数 \(f(\mathbf{x}) = \mathbf{b}^\top\mathbf{x} + a\) 同时是凸的和凹的——它是凸函数和凹函数的”交界”。

二次函数 \(f(\mathbf{x}) = \mathbf{x}^\top\mathbf{Q}\mathbf{x} + \mathbf{b}^\top\mathbf{x} + a\)，若 \(\mathbf{Q} \succeq 0\)（半正定）则为凸函数。

任意范数 \(f(\mathbf{x}) = \|\mathbf{x}\|\)，无论是 \(L^1\)、\(L^2\) 还是 \(L^\infty\)，都是凸函数（由三角不等式可证，方法与证明 \(|x|\) 凸性完全类似）。

最小二乘损失是凸的！因为 \(f(\mathbf{w}) = \|\mathbf{A}\mathbf{w} - \mathbf{y}\|^2 = \mathbf{w}^\top(\mathbf{A}^\top\mathbf{A})\mathbf{w} - 2\mathbf{y}^\top\mathbf{A}\mathbf{w} + \|\mathbf{y}\|^2\)，其中 \(\mathbf{Q} = \mathbf{A}^\top\mathbf{A}\) 始终半正定。

3.6 凸函数的运算保持性

Erhaltung der Konvexität bei Operationen

设 \(f_1, f_2, \ldots\) 是凸函数，则以下函数也是凸的：

(i) 非负加权和：\(f(\mathbf{x}) = c_1 f_1(\mathbf{x}) + c_2 f_2(\mathbf{x})\)，其中 \(c_1, c_2 \geq 0\)

(ii) 逐点最大值：\(f(\mathbf{x}) = \max_i f_i(\mathbf{x})\)

(iii) 逐点极限：\(f(\mathbf{x}) = \lim_{n \to \infty} f_n(\mathbf{x})\)（若极限存在）

Tip🔍 证明 (ii) 逐点最大值（点击展开）

设 \(\mathbf{x}, \mathbf{y} \in \mathcal{X}\)，\(\lambda \in [0, 1]\)。对每个 \(i\)：

\[ f_i\bigl(\lambda\mathbf{x} + (1-\lambda)\mathbf{y}\bigr) \leq \lambda f_i(\mathbf{x}) + (1-\lambda) f_i(\mathbf{y}) \leq \lambda \max_j f_j(\mathbf{x}) + (1-\lambda) \max_j f_j(\mathbf{y}) \]

对所有 \(i\) 取最大值，得到

\[ \max_i f_i\bigl(\lambda\mathbf{x} + (1-\lambda)\mathbf{y}\bigr) \leq \lambda f(\mathbf{x}) + (1-\lambda)f(\mathbf{y}) \quad \square \]

4.1 二阶可微凸函数的三重刻画

Dreifache Charakterisierung zweimal differenzierbarer konvexer Funktionen

设 \(\mathcal{X} \subseteq \mathbb{R}^n\) 是开凸集，\(f : \mathcal{X} \to \mathbb{R}\) 二阶连续可微。则以下三个命题等价：

(i) 弦条件（定义本身）

\[ f\bigl(\lambda\mathbf{x} + (1-\lambda)\mathbf{y}\bigr) \leq \lambda f(\mathbf{x}) + (1-\lambda) f(\mathbf{y}) \quad \forall\, \mathbf{x}, \mathbf{y},\; \lambda \in [0,1] \]

(ii) 切线条件（一阶条件）

\[ f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^\top (\mathbf{y} - \mathbf{x}) \quad \forall\, \mathbf{x}, \mathbf{y} \]

(iii) Hesse 矩阵条件（二阶条件）

\[ \mathbf{H}_f(\mathbf{x}) \succeq 0 \quad \forall\, \mathbf{x} \in \mathcal{X} \]

4.2 次梯度：不可微时的替代方案

Subgradient: Die Alternative bei Nicht-Differenzierbarkeit

像 \(f(x) = |x|\) 这样的凸函数在 \(x = 0\) 处不可微。但凸性保证了一个更一般的性质：对每个 \(\mathbf{x}\)，存在向量 \(\mathbf{v}\)，使得

\[ f(\mathbf{y}) \geq f(\mathbf{x}) + \mathbf{v}^\top(\mathbf{y} - \mathbf{x}) \quad \forall\, \mathbf{y} \in \mathcal{X} \]

满足此条件的 \(\mathbf{v}\) 称为次梯度（Subgradient / Subgradient）。在可微点，次梯度就是梯度；在不可微点，可能有多个次梯度。次梯度对于 LASSO 回归、分位数回归等不可微优化问题至关重要。

5.1 定理陈述

Formulierung des Satzes

Jensen 不等式把凸性的定义从 2 个点推广到了任意多个点：

设 \(f : \mathcal{X} \to \mathbb{R}\) 是凸函数，\(\mathbf{x}_1, \ldots, \mathbf{x}_N \in \mathcal{X}\)，\(w_1, \ldots, w_N \geq 0\)，\(\sum w_i = 1\)，则

\[ \boxed{f\!\left(\sum_{i=1}^{N} w_i \mathbf{x}_i\right) \leq \sum_{i=1}^{N} w_i \, f(\mathbf{x}_i)} \]

用一句话概括：“先平均再求函数值” \(\leq\) “先求函数值再平均”。

用期望的语言重写（取 \(w_i = \mathbb{P}(X = x_i)\)）：

\[ f\bigl(\mathbb{E}[X]\bigr) \leq \mathbb{E}\bigl[f(X)\bigr] \]

方差非负这个”显然”的事实，其实是凸性的直接推论！

6.1 黄金定理：局部即全局

Der goldene Satz: Lokal ist global

这是凸优化中最重要的定理，也是凸性对机器学习最核心的贡献。

定理：设 \(f : \mathbb{R}^n \to \mathbb{R}\) 是可微凸函数，则以下命题等价：

(i) \(\mathbf{x}^\star\) 是 \(f\) 的全局最小值点

(ii) \(\nabla f(\mathbf{x}^\star) = \mathbf{0}\)

这意味着两个惊人的推论：(1) 每个临界点都是全局最小值点（不存在”假的”局部极小值）；(2) 梯度下降只要找到 \(\nabla f = \mathbf{0}\) 的点，就一定是全局最优。

Tip🔍 证明（点击展开）

(i)⟹(ii)：若 \(\mathbf{x}^\star\) 是全局最小值点，它一定是局部最小值点，故 \(\nabla f(\mathbf{x}^\star) = \mathbf{0}\)（微积分的必要条件）。

(ii)⟹(i)：若 \(\nabla f(\mathbf{x}^\star) = \mathbf{0}\)，由切线条件（凸函数的一阶条件），对任意 \(\mathbf{y}\)：

\[ f(\mathbf{y}) \geq f(\mathbf{x}^\star) + \underbrace{\nabla f(\mathbf{x}^\star)^\top}_{= \mathbf{0}^\top}(\mathbf{y} - \mathbf{x}^\star) = f(\mathbf{x}^\star) \quad \square \]

6.2 严格凸性：保证唯一解

Strikte Konvexität: Eindeutigkeit der Lösung garantiert

定义：若对所有 \(\mathbf{x} \neq \mathbf{y}\) 及所有 \(\lambda \in (0, 1)\)，严格不等式

\[ f\bigl(\lambda\mathbf{x} + (1-\lambda)\mathbf{y}\bigr) < \lambda f(\mathbf{x}) + (1-\lambda) f(\mathbf{y}) \]

成立，则称 \(f\) 是严格凸的（strikt konvex / strictly convex）。

命题：若 \(f\) 是严格凸的，则 \(f\) 至多有一个最小值点。

凸性保证”局部即全局”，严格凸性进一步保证”解是唯一的”。

凸性类型 / Konvexitätstyp 最小值点 / Minimumstelle 例子 / Beispiel
凸（非严格） 可能有多个（甚至一个”平底”） \(f(x) = \max(0, x-1)\) 在 \((-\infty, 1]\) 上
严格凸 至多一个 \(f(x) = x^2\)，岭回归

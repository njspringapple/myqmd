迭代方法与概率方法：从特征值到矩阵草图

1 前置知识回顾

在开始之前，确认你对以下概念有基本了解：

  • 算法复杂度：大 \(O\) 记号衡量运行时间随问题规模的增长
  • 稳定性与条件数：数值求解的误差分析
  • 正交矩阵：\(\mathbf{Q}^\top\mathbf{Q} = \mathbf{I}\)，不放大误差
  • 谱范数 \(\|\mathbf{A}\|_2\)：矩阵的最大奇异值
  • 线性方程组：LU 分解、Cholesky 分解（上一章）
  • 线性代数：特征值与特征向量，相似变换 \(\mathbf{B} = \mathbf{Q}\mathbf{A}\mathbf{Q}^{-1}\)
  • 概率论：随机向量、期望、方差
  • 分析：数列收敛性，几何级数 \(\sum_k p^k\)（当 \(|p| < 1\) 时收敛）

上一章建立了矩阵分解（LU、Cholesky）的框架，可以精确求解线性方程组。本章介绍两类补充方法：

  • 迭代方法：以牺牲精度换取更短的运行时间
  • 概率方法：用随机的小问题替代大问题，高概率得到近似解

这两个方向在现代数值线性代数和大规模机器学习中都极为重要。

# 辅助函数
vnorm <- function(x) sqrt(sum(x^2))
fnorm <- function(A) sqrt(sum(A^2))

# 向量夹角（角度）
vec_angle <- function(x, y) {
  cos_val <- sum(x * y) / (vnorm(x) * vnorm(y))
  cos_val <- pmax(-1, pmin(1, cos_val))  # 防止数值误差超出 [-1,1]
  acos(cos_val) * 180 / pi
}

# 前向替换（下三角系统 Ly = b）
forward_substitution <- function(L, b) {
  n <- nrow(L)
  y <- numeric(n)
  y[1] <- b[1] / L[1, 1]
  for (i in 2:n) {
    y[i] <- (b[i] - sum(L[i, 1:(i-1)] * y[1:(i-1)])) / L[i, i]
  }
  y
}

2 动机：直接方法的局限性

2.1 规模的瓶颈

上一章的 LU 和 Cholesky 分解都是 \(O(n^3)\) 的算法。对于 \(n = 1000\)，这意味着约 \(10^9\) 次运算——可以接受。但现实问题的规模往往大得多：

  • Google PageRank：\(n \approx 10^{10}\) 个网页
  • 基因表达分析：\(p = 10^4\) 到 \(10^6\) 个基因
  • 图像处理：每张图片有数百万像素
  • 偏微分方程离散化：得到 \(n > 10^6\) 的大型稀疏矩阵

对这些问题，\(O(n^3)\) 的直接方法根本不可能。迭代方法则可以利用矩阵的稀疏性——只需做矩阵-向量乘法，复杂度降至 \(O(\text{nnz}(\mathbf{A}))\)（非零元素数量）。

2.2 只需少数特征值

在 PCA 中，我们通常只关心最大的 \(k \ll p\) 个主成分。Lanczos 迭代只需 \(O(kp^2)\) 而非 \(O(p^3)\)——当 \(p = 100\,000\), \(k = 10\) 时，快约 10,000 倍！

本章的核心信息：直接方法精确但昂贵，迭代方法和概率方法用少量误差换取巨大的速度提升。

3 特征值问题

3.1 为什么特征值没有直接方法？

求特征值的直接思路是求特征多项式 \(\det(\mathbf{A} - \lambda\mathbf{I}) = 0\) 的根。但这遇到了数学上的根本性障碍：

Abel-Ruffini 定理：5 次及以上的多项式方程没有一般的代数解公式。因此对于 \(n \geq 5\)，不存在基于特征多项式的封闭形式算法。即使不用特征多项式，直接方法也不稳定、代价高昂（\(O(n^4)\) 或更差）。

3.2 幂方法

幂方法（von Mises 迭代）通过反复用矩阵 \(\mathbf{A}\) 乘以一个向量，提取最大（绝对值意义下分离的！）特征值 \(\lambda_1\) 及对应特征向量 \(\mathbf{v}_1\)。

3.2.1 算法

从随机初始向量 \(\mathbf{x}^{(0)}\) 出发，迭代：

\[ \mathbf{x}^{(k)} = \frac{\mathbf{A}\mathbf{x}^{(k-1)}}{\|\mathbf{A}\mathbf{x}^{(k-1)}\|} \]

则 \(\lambda^{(k)} = \|\mathbf{A}\mathbf{x}^{(k)}\| \xrightarrow{k \to \infty} \lambda_1\)，\(\mathbf{x}^{(k)} \xrightarrow{k \to \infty} \pm\mathbf{v}_1\)。

3.2.2 为什么有效？代数推导

任何初始向量都可以写成特征向量的线性组合：\(\mathbf{x}^{(0)} = \sum_{i=1}^n c_i \mathbf{v}_i\)。

利用 \(\mathbf{A}^k\mathbf{v}_i = \lambda_i^k\mathbf{v}_i\)，可得：

\[ \mathbf{A}^k\mathbf{x}^{(0)} = \lambda_1^k\left(c_1\mathbf{v}_1 + \sum_{i=2}^n c_i\left(\frac{\lambda_i}{\lambda_1}\right)^k\mathbf{v}_i\right) \]

当 \(k \to \infty\) 时，由于 \(|\lambda_i/\lambda_1| < 1\)（对所有 \(i \geq 2\)），其他分量趋于零：

\[ \frac{\mathbf{A}^k\mathbf{x}^{(0)}}{\|\mathbf{A}^k\mathbf{x}^{(0)}\|} \approx \frac{\lambda_1^k c_1 \mathbf{v}_1}{|\lambda_1^k c_1|\|\mathbf{v}_1\|} = \pm\mathbf{v}_1 \]

# 幂方法实现
power_method <- function(A, x0 = NULL, max_iter = 100, tol = 1e-12) {
  n <- nrow(A)
  if (is.null(x0)) { set.seed(42); x0 <- rnorm(n) }
  x <- x0 / vnorm(x0)

  lambda_hist <- numeric(max_iter)
  x_hist      <- matrix(0, n, max_iter)

  for (k in 1:max_iter) {
    Ax <- A %*% x
    lambda_new <- as.numeric(t(x) %*% Ax)   # Rayleigh 商
    x_new <- Ax / vnorm(Ax)

    lambda_hist[k] <- lambda_new
    x_hist[, k]    <- as.numeric(x_new)

    if (k > 1 && abs(lambda_new - lambda_hist[k-1]) < tol) {
      return(list(lambda = lambda_new, vector = as.numeric(x_new),
                  lambda_hist = lambda_hist[1:k],
                  x_hist = x_hist[, 1:k, drop = FALSE],
                  iterations = k))
    }
    x <- x_new
  }
  list(lambda = lambda_hist[max_iter], vector = as.numeric(x),
       lambda_hist = lambda_hist, x_hist = x_hist, iterations = max_iter)
}

3.4 相似变换与迭代特征值方法

3.4.1 核心定理

定理：若 \(\mathbf{B} = \mathbf{Q}\mathbf{A}\mathbf{Q}^{-1}\)（相似变换），则：

  1. \(\mathbf{A}\) 和 \(\mathbf{B}\) 有完全相同的特征值（行列式、迹）
  2. 若 \(\mathbf{y}\) 是 \(\mathbf{B}\) 的特征向量，则 \(\mathbf{x} = \mathbf{Q}^{-1}\mathbf{y}\) 是 \(\mathbf{A}\) 对应同一特征值的特征向量

证明：

\[ \mathbf{B}\mathbf{y} = \lambda\mathbf{y} \implies \mathbf{Q}\mathbf{A}\mathbf{Q}^{-1}\mathbf{y} = \lambda\mathbf{y} \implies \mathbf{A}\underbrace{\mathbf{Q}^{-1}\mathbf{y}}_{\mathbf{x}} = \lambda\underbrace{\mathbf{Q}^{-1}\mathbf{y}}_{\mathbf{x}} \]

因此，如果我们能通过相似变换把 \(\mathbf{A}\) 变为一个”简单”矩阵（对角或三角矩阵），就可以直接读出特征值！

3.5 QR 迭代

QR 迭代是计算所有特征值/特征向量的主流方法。其思路是：利用 QR 分解进行相似变换，使矩阵逐渐收敛到上三角矩阵（Schur 分解）。

3.5.1 算法定义

\[ \begin{aligned} &\mathbf{A}^{(0)} = \mathbf{A} \\ &\text{对于 } k = 1, 2, \ldots: \\ &\quad \mathbf{A}^{(k-1)} = \mathbf{Q}^{(k)}\mathbf{R}^{(k)} \quad \text{（QR 分解）} \\ &\quad \mathbf{A}^{(k)} = \mathbf{R}^{(k)}\mathbf{Q}^{(k)} = \left(\mathbf{Q}^{(k)}\right)^\top \mathbf{A}^{(k-1)}\mathbf{Q}^{(k)} \quad \text{（相似变换！）} \end{aligned} \]

关键点：\(\mathbf{A}^{(k)}\) 始终与 \(\mathbf{A}\) 相似，故特征值不变。当 \(k \to \infty\) 时，\(\mathbf{A}^{(k)}\) 收敛到上三角矩阵，对角线上即为特征值。

# QR 迭代实现
qr_iteration <- function(A, max_iter = 200, tol = 1e-12) {
  n <- nrow(A)
  Ak <- A
  Q_total <- diag(n)
  eig_hist <- matrix(0, max_iter, n)

  for (k in 1:max_iter) {
    qr_k <- qr(Ak)
    Q_k  <- qr.Q(qr_k)
    R_k  <- qr.R(qr_k)

    Ak      <- R_k %*% Q_k           # 相似变换
    Q_total <- Q_total %*% Q_k       # 累积正交变换
    eig_hist[k, ] <- diag(Ak)

    # 收敛：下三角部分趋于零
    if (fnorm(Ak[lower.tri(Ak)]) < tol) {
      return(list(
        eigenvalues  = sort(diag(Ak), decreasing = TRUE),
        eigenvectors = Q_total,
        A_final      = Ak,
        history      = eig_hist[1:k, , drop = FALSE],
        iterations   = k
      ))
    }
  }
  list(eigenvalues = sort(diag(Ak), decreasing = TRUE),
       eigenvectors = Q_total, A_final = Ak,
       history = eig_hist, iterations = max_iter)
}

3.5.2 QR 迭代的数学本质

定理：定义 \(\mathbf{Q}_k = \mathbf{Q}^{(1)}\mathbf{Q}^{(2)}\cdots\mathbf{Q}^{(k)}\)，\(\mathbf{R}_k = \mathbf{R}^{(k)}\cdots\mathbf{R}^{(1)}\)，则

\[ \mathbf{A}^k = \mathbf{Q}_k\mathbf{R}_k \]

证明（归纳法）：

  • 基础 \((k=1)\)：\(\mathbf{A} = \mathbf{Q}^{(1)}\mathbf{R}^{(1)} = \mathbf{Q}_1\mathbf{R}_1\) ✓
  • 归纳步骤：利用关键关系 \(\mathbf{R}^{(i)}\mathbf{Q}^{(i)} = \mathbf{Q}^{(i+1)}\mathbf{R}^{(i+1)}\)（即 \(\mathbf{A}^{(i)} = \mathbf{A}^{(i)}\)），可推得 \(\mathbf{A}^{k+1} = \mathbf{Q}_{k+1}\mathbf{R}_{k+1}\) ✓

这说明 QR 迭代就是在迭代计算 \(\mathbf{A}^k\) 的 QR 分解，\(\mathbf{Q}_k\) 的各列相当于对所有单位向量同时做幂方法，因此 \(\mathbf{Q}_k \to \mathbf{V}\)（特征向量矩阵），从而：

\[ \mathbf{A}^{(k)} = \mathbf{Q}_k^\top\mathbf{A}\mathbf{Q}_k \xrightarrow{k\to\infty} \mathbf{V}^\top\mathbf{A}\mathbf{V} = \mathbf{V}^\top\mathbf{V}\mathbf{\Lambda}\mathbf{V}^\top\mathbf{V} = \mathbf{\Lambda} \]

4.1 不动点迭代的直觉

我们要求解 \(\mathbf{Ax} = \mathbf{b}\)。直接法（LU）精确但需要 \(O(n^3)\)。迭代法的思路是逐步改进近似解。

残差：\(\mathbf{r}^{(k)} = \mathbf{b} - \mathbf{A}\mathbf{x}^{(k)}\) 衡量当前估计有多差。当 \(\mathbf{r}^{(k)} = \mathbf{0}\) 时找到了精确解。

不动点迭代（校正步）：

\[ \mathbf{x}^{(k)} = \mathbf{x}^{(k-1)} + \underbrace{\mathbf{C}\mathbf{r}^{(k-1)}}_{\text{校正步}} = \mathbf{x}^{(k-1)} + \mathbf{C}(\mathbf{b} - \mathbf{A}\mathbf{x}^{(k-1)}) \]

矩阵 \(\mathbf{C}\) 决定校正方式和强度。理想情况下 \(\mathbf{C} \approx \mathbf{A}^{-1}\)——但计算 \(\mathbf{A}^{-1}\) 正是我们想避免的！

类比：这与梯度下降完全相同的结构：每步沿残差（负梯度）方向走一步。

4.2 收敛定理

定理：设 \(\mathbf{x}\) 是 \(\mathbf{Ax} = \mathbf{b}\) 的唯一解，令 \(\rho := \|\mathbf{I} - \mathbf{CA}\|_2\)。若 \(\rho < 1\)，则对所有 \(k \geq 1\)：

\[ \|\mathbf{x}^{(k)} - \mathbf{x}\|_2 \leq \rho^k\|\mathbf{x}^{(0)} - \mathbf{x}\|_2 \]

特别地，\(\lim_{k\to\infty}\mathbf{x}^{(k)} = \mathbf{x}\)。

证明：令 \(\mathbf{B} = \mathbf{I} - \mathbf{CA}\)，则迭代写为 \(\mathbf{x}^{(k)} = \mathbf{B}\mathbf{x}^{(k-1)} + \mathbf{Cb}\)。

首先验证 \(\mathbf{x}\) 是不动点：

\[ \mathbf{Bx} + \mathbf{Cb} = (\mathbf{I} - \mathbf{CA})\mathbf{x} + \mathbf{Cb} = \mathbf{x} - \mathbf{C}\underbrace{(\mathbf{Ax} - \mathbf{b})}_{=\,\mathbf{0}} = \mathbf{x} \]

因此：

\[ \|\mathbf{x}^{(k)} - \mathbf{x}\|_2 = \|\mathbf{B}(\mathbf{x}^{(k-1)} - \mathbf{x})\|_2 \leq \|\mathbf{B}\|_2\|\mathbf{x}^{(k-1)} - \mathbf{x}\|_2 = \rho\|\mathbf{x}^{(k-1)} - \mathbf{x}\|_2 \]

迭代此不等式即得结论。

4.2.1 需要多少次迭代？

求解 \(\rho^k \|\mathbf{x}^{(0)} - \mathbf{x}\| \leq \epsilon\)：

\[ k \geq \frac{\log(e_0/\epsilon)}{-\log(\rho)}, \quad e_0 = \|\mathbf{x}^{(0)} - \mathbf{x}\| \]

因此 \(k = O(\log(1/\epsilon))\)——误差级数收敛，精度每提高一个数量级只需约 \(\log(10)/(-\log\rho)\) 步。

4.3.1 Richardson 迭代

选择 \(\mathbf{C} = \gamma\mathbf{I}_n\)（步长最简单）：

\[ \mathbf{x}^{(k)} = \mathbf{x}^{(k-1)} + \gamma(\mathbf{b} - \mathbf{A}\mathbf{x}^{(k-1)}) \]

最优步长（当 \(\mathbf{A}\) 的特征值已知）：

\[ \gamma^* = \frac{2}{\lambda_{\min} + \lambda_{\max}}, \qquad \rho^* = \frac{\kappa - 1}{\kappa + 1} \]

其中 \(\kappa = \lambda_{\max}/\lambda_{\min}\) 是条件数。条件数越大，\(\rho^*\) 越接近 1，收敛越慢。

# 三种方法的统一对比
run_iterative <- function(A, b, method = "richardson",
                          gamma = NULL, max_iter = 300, tol = 1e-11) {
  n <- length(b)
  x <- rep(0, n)
  x_true <- solve(A, b)
  errors <- numeric(max_iter)

  for (k in 1:max_iter) {
    r <- b - A %*% x
    if (method == "richardson") {
      x <- x + gamma * r
    } else if (method == "jacobi") {
      x <- x + (1 / diag(A)) * r
    } else if (method == "gauss-seidel") {
      L_lower <- A; L_lower[upper.tri(L_lower)] <- 0
      x <- forwardsolve(L_lower, b - (A - L_lower) %*% x)
    }
    errors[k] <- vnorm(x - x_true) / vnorm(x_true)
    if (errors[k] < tol) return(list(x = x, errors = errors[1:k]))
  }
  list(x = x, errors = errors)
}

5.3 随机嵌入定理

定理：设 \(\mathbf{S} \in \mathbb{R}^{m \times n}\) 的行向量 \(\mathbf{s}_i\) 独立同分布，满足 \(\mathbb{E}[\mathbf{s}_i\mathbf{s}_i^\top] = \mathbf{I}_n/m\) 且 \(\mathbb{E}[\|\mathbf{s}_i\|^4] \leq K/m^2\)，则以至少 \(1 - \delta\) 的概率：

\[ (1 - \epsilon)\|\mathbf{x}\|^2 \leq \|\mathbf{Sx}\|^2 \leq (1 + \epsilon)\|\mathbf{x}\|^2 \quad \text{对所有 } \mathbf{x} \in \mathbb{R}^n \]

其中 \(\epsilon = \sqrt{K/(\delta m)}\)。

证明思路（用 Markov 不等式）：

  1. 不失一般性设 \(\|\mathbf{x}\| = 1\)
  2. 计算期望：\(\mathbb{E}[\|\mathbf{Sx}\|^2] = m \cdot \mathbf{x}^\top\mathbb{E}[\mathbf{s}_1\mathbf{s}_1^\top]\mathbf{x} = \|\mathbf{x}\|^2 = 1\)
  3. 估计方差：\(\text{Var}[\|\mathbf{Sx}\|^2] \leq m\mathbb{E}[\|\mathbf{s}_1\|^4] \leq K/m\)
  4. 由 Markov 不等式：\(\mathbb{P}(|\|\mathbf{Sx}\|^2 - 1| > \epsilon) \leq K/(\epsilon^2 m)\)
  5. 令右端等于 \(\delta\)，解出 \(\epsilon = \sqrt{K/(\delta m)}\)

三种典型草图矩阵：

  • 高斯：\(\mathbf{s}_i \sim \mathcal{N}(\mathbf{0}, \mathbf{I}_n/m)\)，理论最优，\(K = 3\)
  • Rademacher：\(s_{ij} = \pm 1/\sqrt{m}\) 各以 \(1/2\) 概率取值，\(K = 1\)，与高斯同样好
  • 子采样：\(\mathbf{s}_i = \sqrt{n/m}\,\mathbf{e}_{K_i}\)，\(K_i \sim \text{Unif}\{1,\ldots,n\}\)，极快 \(O(m)\)

8.1 知识地图

数值线性代数（迭代方法 & 概率方法）
│
├── 特征值问题
│   ├── 为什么没有直接法？
│   │   └── Abel-Ruffini 定理：n≥5 时无代数公式
│   │
│   ├── 幂方法（von Mises）
│   │   ├── 思想：A^k x → 主特征向量方向
│   │   ├── 每步：O(nnz(A))（仅矩阵-向量乘法）
│   │   ├── 收敛率：|λ₂/λ₁|
│   │   └── 适用：最大 1 个特征对
│   │
│   ├── QR 迭代
│   │   ├── 思想：相似变换收敛到 Schur 分解（上三角）
│   │   ├── 等价：对所有方向同时做幂方法
│   │   ├── 定理：A^k = Q_k R_k
│   │   └── 适用：所有特征值（n 不太大）
│   │
│   ├── Lanczos 迭代（提及）
│   │   └── 稀疏大矩阵的少数极端特征值，O(m·nnz)
│   │
│   └── 应用：PageRank（幂方法），PCA（特征分解/SVD）
│
├── 线性方程组的迭代求解
│   ├── 不动点迭代：x(k) = x(k-1) + C(b - Ax(k-1))
│   ├── 收敛条件：ρ = ||I - CA||₂ < 1
│   ├── 误差界：||x(k) - x|| ≤ ρᵏ ||x(0) - x||
│   ├── 收敛步数：k = O(log(1/ε))
│   ├── 单步代价：O(n²)，总代价 O(n² log(1/ε))
│   │              vs 直接法 O(n³)
│   │
│   ├── Richardson：C = γI，最优 γ* = 2/(λ_min + λ_max)
│   ├── Jacobi：   C = diag(A)⁻¹
│   └── Gauss-Seidel：C⁻¹ = 下三角(A)
│
└── 概率方法：矩阵草图
    ├── 思想：S ∈ R^{m×n}（m ≪ n）随机投影降维
    ├── 定理：(1-ε)||x||² ≤ ||Sx||² ≤ (1+ε)||x||²
    ├── 维度：m = K/(δε²)
    │
    ├── 高斯：K=3，理论最优
    ├── Rademacher：K=1，与高斯同样好，更快生成
    ├── 子采样：O(m)，对结构化矩阵最快
    └── SRHT：O(n log m)，最先进
